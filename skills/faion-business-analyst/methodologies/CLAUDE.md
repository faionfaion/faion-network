# Methodologies Folder

## Overview

Contains 18 detailed BA methodology files (M-BA-001 to M-BA-018) based on BABOK v3 Knowledge Areas. Each file provides frameworks, templates, and examples for specific BA activities.

## File Structure

Each methodology file follows a consistent structure:

```markdown
# M-BA-XXX: [Name]

## Metadata
- Category, Difficulty, Tags, Agent

## Problem
- What issue this methodology solves

## Framework
- Step-by-step approach

## Templates
- Ready-to-use templates (markdown)

## Examples
- Practical application examples

## Common Mistakes
- Pitfalls to avoid

## Next Steps
- Connection to other methodologies

## References
- BABOK Guide v3, IIBA Standards
```
