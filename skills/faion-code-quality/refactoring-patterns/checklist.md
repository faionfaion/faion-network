# Checklist

Step-by-step checklist for applying this methodology.

## Prerequisites

- [ ] Understand the methodology concepts (read README.md)
- [ ] Review templates and examples
- [ ] Identify applicable scenarios in your project

## Implementation Steps

### 1. Planning

- [ ] Define goals and success criteria
- [ ] Identify affected components
- [ ] Review team understanding

### 2. Preparation

- [ ] Set up necessary tools
- [ ] Review existing code/documentation
- [ ] Plan incremental approach

### 3. Execution

- [ ] Follow methodology patterns
- [ ] Apply templates where appropriate
- [ ] Document decisions

### 4. Verification

- [ ] Review against methodology principles
- [ ] Get team feedback
- [ ] Measure results against criteria

### 5. Iteration

- [ ] Identify improvements
- [ ] Update documentation
- [ ] Share learnings with team

## Common Pitfalls

- Skipping planning phase
- Trying to apply everything at once
- Not adapting to context
- Ignoring team feedback

## Success Indicators

- Methodology applied consistently
- Team understands the approach
- Measurable improvements visible
- Documentation up to date
