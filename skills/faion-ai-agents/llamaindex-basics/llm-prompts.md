# LLM Prompts

Prompts for AI-assisted implementation of this methodology.

## Code Generation Prompts

### Initial Implementation

```
You are an expert in [METHODOLOGY]. Generate a production-ready implementation that:
- Follows best practices
- Includes error handling
- Has comprehensive logging
- Is well-documented

Requirements:
[DESCRIBE REQUIREMENTS]

Constraints:
[DESCRIBE CONSTRAINTS]
```

### Code Review

```
Review the following code for [METHODOLOGY] implementation:

[CODE HERE]

Check for:
- Adherence to best practices
- Security vulnerabilities
- Performance issues
- Code quality and maintainability
- Missing error handling
```

### Optimization

```
Optimize this [METHODOLOGY] implementation for:
- Performance
- Scalability
- Maintainability

Current implementation:
[CODE HERE]

Provide specific improvements with explanations.
```

## Documentation Prompts

```
Generate comprehensive documentation for this [METHODOLOGY] implementation:

[CODE HERE]

- Troubleshooting guide
```

### Create Tests

```
Generate comprehensive test cases for this [METHODOLOGY] implementation:

[CODE HERE]

```

## Debugging Prompts

### Troubleshooting

```
I'm experiencing [ISSUE] with this [METHODOLOGY] implementation:

[CODE/ERROR HERE]

Help me:
1. Identify the root cause
2. Provide a fix
3. Explain how to prevent this in the future
```

### Performance Analysis

```
Analyze the performance of this [METHODOLOGY] implementation:

[CODE HERE]

Identify bottlenecks and suggest optimizations.
```

## Learning Prompts

### Concept Explanation

```
Explain [CONCEPT] in [METHODOLOGY] as if I'm:
- A beginner (simple terms)
- Intermediate (with examples)
- Advanced (deep dive with edge cases)
```

### Comparison

```
Compare [METHODOLOGY A] vs [METHODOLOGY B] for [USE CASE]:
- Pros and cons
- Performance characteristics
- When to use each
- Migration considerations
```

## Tips for Using These Prompts

1. **Be Specific**: Replace placeholders with actual details
2. **Provide Context**: Include relevant background information
3. **Iterate**: Refine prompts based on initial responses
4. **Validate**: Always review AI-generated code carefully
5. **Learn**: Use prompts to understand, not just copy-paste
