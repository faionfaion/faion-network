# Link Building Checklist

## Phase 1: Preparation & Strategy

### Content Audit
- [ ] Audit existing content for link-worthiness
- [ ] Identify 3-5 potential "link magnet" assets
- [ ] Document current backlinks (Ahrefs, Moz)
- [ ] Analyze competitor backlink profiles
- [ ] Identify high-value, underlinked content

### Content Creation Strategy
- [ ] Decide on link magnet type (research/tool/guide/data)
- [ ] Plan original research topic (survey/analysis)
- [ ] Define target audience for outreach
- [ ] Set link acquisition target (# of links)
- [ ] Estimate effort required

### Tool & Account Setup
- [ ] Set up Ahrefs account (or Semrush/Moz)
- [ ] Set up Hunter.io account (email finding)
- [ ] Set up email tracking tool (optional)
- [ ] Create link building tracking spreadsheet
- [ ] Install Ahrefs SEO toolbar

## Phase 2: Create Link-Worthy Content

### Asset Development
- [ ] Create or update link magnet content
- [ ] Ensure comprehensive, unique value
- [ ] Add visuals (charts, infographics, data)
- [ ] Optimize for sharing (buttons, embeds)
- [ ] Create easy-to-embed version (if applicable)
- [ ] Write dedicated blog post explaining asset
- [ ] Create easy shareable URL (short domain)

### Promotion Readiness
- [ ] Write 1-2 paragraph asset description
- [ ] Create key statistics/findings summary
- [ ] Prepare fact sheet or infographic
- [ ] Design social media graphics
- [ ] Create LinkedIn post
- [ ] Create Twitter thread

## Phase 3: Find Link Opportunities

### Competitor Analysis
- [ ] Export competitor backlinks (Ahrefs)
- [ ] Filter by domain authority (40+)
- [ ] Remove non-relevant sites
- [ ] Identify most common linking sites
- [ ] Identify industry leaders with many backlinks
- [ ] Create priority list (DA 50+, 40-49, 30-39)

### Resource Page Prospecting
- [ ] Search "[topic] + resources" in Google
- [ ] Search "[topic] + links" in Google
- [ ] Search "[topic] + useful" in Google
- [ ] Search "[topic] + directory" in Google
- [ ] Extract URLs and analyze
- [ ] Filter by domain authority

### Broken Link Hunting
- [ ] Identify relevant "authority" pages in niche
- [ ] Use Ahrefs Broken Links tool
- [ ] Manually check links on high-DA pages
- [ ] Document broken link URLs
- [ ] Note what content it was linking to

### HARO & Journalist Opportunities
- [ ] Sign up for HARO (Help A Reporter Out)
- [ ] Monitor daily queries
- [ ] Respond to 2-3 relevant queries daily
- [ ] Track which responses get featured

### Build Prospect Database
- [ ] Create master prospect list (100+ sites)
- [ ] Segment by opportunity type
- [ ] Segment by domain authority
- [ ] Segment by difficulty
- [ ] Add contact information for each

## Phase 4: Outreach Execution

### Research & Personalization Prep
- [ ] For each target, read 2-3 recent articles
- [ ] Identify author/editor (LinkedIn, staff page)
- [ ] Note specific content relevant to your asset
- [ ] Personalize first line for each outreach

### Email Outreach
- [ ] Write 2-3 outreach templates
- [ ] Customize for audience (journalists, bloggers, etc.)
- [ ] Prepare personalization variables
- [ ] Schedule sending in batches (50 max/day)
- [ ] Send waves of 50 emails
- [ ] Log all sent in tracking sheet

### Response Management
- [ ] Check email daily for responses
- [ ] Respond within 24 hours
- [ ] Answer questions about content
- [ ] Provide additional supporting materials if needed
- [ ] Follow up if link promised (gentle)

### Follow-up Process
- [ ] Wait 7 days after initial send
- [ ] Send single follow-up (brief, polite)
- [ ] Wait another 7 days, then move on
- [ ] Do not follow up more than once

## Phase 5: Track & Analyze Results

### Link Monitoring
- [ ] Check for new backlinks daily (first week)
- [ ] Verify links are dofollow (Google Search Console)
- [ ] Note anchor text used
- [ ] Document domain authority of linking sites
- [ ] Confirm link placement (context, above/below fold)

### Campaign Tracking
- [ ] Update spreadsheet with responses
- [ ] Mark "link acquired" vs "no response"
- [ ] Calculate response rate (%)
- [ ] Calculate link acquisition rate (%)
- [ ] Average domain authority of new links
- [ ] Total new referring domains

### Metric Analysis
- [ ] Response rate: Target 10%+ (20%+ is great)
- [ ] Link acquisition rate: Target 5%+ (10%+ is great)
- [ ] Average DR of new links: Target 40+ (60+ is great)
- [ ] New referring domains: Target 5+/month (20+/month is great)
- [ ] Ranking improvements for target keywords

## Phase 6: Optimization & Scaling

### What Worked Analysis
- [ ] Identify which outreach templates had highest response
- [ ] Note which prospects/types were most responsive
- [ ] Document best performing anchor texts
- [ ] Analyze common link sources (resource pages, guest post, etc.)

### Refinement
- [ ] Update templates based on learnings
- [ ] Focus on highest-converting opportunities
- [ ] Increase volume on successful tactics
- [ ] Phase out low-performing approaches
- [ ] Test new angles/assets

### Internal Linking Strategy
- [ ] Audit internal linking on main website
- [ ] Link new posts to old relevant content
- [ ] Update old posts to link to new content
- [ ] Create pillar page linking structure
- [ ] Ensure keyword-rich anchor text
- [ ] Audit for orphan pages (no internal links)

### Scalability
- [ ] Document repeatable process
- [ ] Create SOP for future campaigns
- [ ] Set up monthly link building target
- [ ] Plan next link magnet content
- [ ] Establish ongoing outreach rhythm

## Phase 7: Monthly Maintenance

### Weekly Tasks
- [ ] Monitor new backlinks
- [ ] Check link quality (not spam)
- [ ] Respond to outreach inquiries
- [ ] Follow up if needed

### Monthly Tasks
- [ ] Review link acquisition metrics
- [ ] Analyze ranking changes
- [ ] Plan next campaign
- [ ] Update outreach lists
- [ ] Optimize underperforming content
- [ ] Report on results

### Ongoing
- [ ] Maintain link building outreach (20-50/month minimum)
- [ ] Create linkable assets quarterly
- [ ] Monitor competitor link activity
- [ ] Adapt strategy as market changes
