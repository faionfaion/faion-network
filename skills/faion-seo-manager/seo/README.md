## Core Principles

### 1. Dual Optimization Strategy

```
Traditional SEO (Google, Bing)
        ↓
    Your Content
        ↓
AI Search (ChatGPT, Perplexity, Claude)
```

**Both need:** Clear structure, authoritative content, proper markup, fast loading.

### 2. E-E-A-T Framework

- **Experience** → First-hand knowledge, case studies
- **Expertise** → Author credentials, depth of content
- **Authoritativeness** → Backlinks, citations, brand mentions
- **Trustworthiness** → HTTPS, contact info, privacy policy

## Quick Workflow

### For Technical SEO Audit

1. Read [seo-basics.md](seo-basics.md)
2. Use Technical SEO Checklist
3. Check Core Web Vitals
4. Validate robots.txt and sitemap.xml

### For Content Optimization

1. Read [seo-techniques.md](seo-techniques.md)
2. Implement schema markup (Article, Breadcrumb, FAQ)
3. Add social meta tags (Open Graph, Twitter Cards)
4. Set up internal linking (pillar-cluster model)

### For AI Search Visibility

1. Read [technical-seo-for-ai.md](technical-seo-for-ai.md)
2. Create llms.txt file
3. Add Organization schema
4. Optimize for entity connections
