# YouTube Strategy - Checklist

## Pre-Implementation

- [ ] Define YouTube channel goals and target audience
- [ ] Create channel branding (art, description, links)
- [ ] Research competitor YouTube strategies
- [ ] Plan content themes and content pillars
- [ ] Create 12-week video content calendar
- [ ] Identify trending topics in niche
- [ ] Plan video series with recurring concepts
- [ ] Create compelling video titles using keyword research

## Implementation Phase

- [ ] Write detailed video descriptions with links
- [ ] Design custom thumbnails for high CTR
- [ ] Add captions to all videos for accessibility
- [ ] Use tags strategically for discoverability
- [ ] Create playlists organizing videos by theme
- [ ] Plan video promotion across other channels
- [ ] Monitor YouTube analytics monthly (views, CTR, retention)
