# Podcast Strategy - Checklist

## Pre-Implementation

- [ ] Define podcast goals (audience building, thought leadership, leads)
- [ ] Choose podcast format (solo, co-hosted, interview, narrative)
- [ ] Research podcast hosting platform (Buzzsprout, Transistor, Podbean)
- [ ] Plan episode topics aligned with ICP pain points
- [ ] Create podcast branding (name, cover art, description)
- [ ] Write 12-week episode outline and guest list
- [ ] Set up recording and editing workflow
- [ ] Create podcast show notes template

## Implementation Phase

- [ ] Establish publishing schedule (weekly, bi-weekly, monthly)
- [ ] Set up podcast distribution to major platforms (Spotify, Apple, Google)
- [ ] Create guest recruitment strategy and outreach template
- [ ] Plan cross-promotion with blog/newsletter
- [ ] Design podcast landing page with episode feeds
- [ ] Track podcast metrics (downloads, listener growth, audience engagement)
- [ ] Plan content repurposing (clips, transcripts, blog posts)
