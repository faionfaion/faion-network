# AI Content Strategy - Templates

## AI Content Prompt Template

```markdown
# Content Brief: [Topic]

## Role
You are a [expert type] helping create content about [topic].

## Constraints
- Avoid generic advice from top 10 results
- Focus on [specific perspective/angle]
- Target audience: [specific ICP]
- Writing level: 6th-8th grade

## Context
- Company: [Your company]
- Product: [Your product]
- Unique angle: [Your differentiation]

## Task
Write a [content type] about [topic] that:
1. Opens with [specific angle]
2. Includes [your proprietary data/insight]
3. References [specific examples you've experienced]
4. Ends with actionable next steps

## Output
Format as [markdown/outline/draft]
Target length: [words/sections]
```

## Content Differentiation Checklist Template

```markdown
# Content Review: Is This Differentiated?

## Generic Content Signals (AVOID)
- [ ] "According to [Industry Report]" (aggregated)
- [ ] "Best practices are..." (generic)
- [ ] "You should..." (surface-level)
- [ ] No specific examples or numbers
- [ ] No opinion or stance taken

## Differentiated Content Signals (INCLUDE)
- [ ] Own research or data: _______________
- [ ] First-hand experience: _______________
- [ ] Expert quote/interview: _______________
- [ ] Specific case study with results: _______________
- [ ] Contrarian take or opinion: _______________
- [ ] Real examples with outcomes: _______________
- [ ] Failed experiments or lessons: _______________
- [ ] E-E-A-T signals present: _______________

## Verdict
- [ ] Ready to publish (differentiated)
- [ ] Needs revision (add proprietary elements)
- [ ] Reject (too generic)
```

## AI + Human Collaboration Workflow

```markdown
## Content Creation Workflow

### Stage 1: AI Research & Outline (AI)
- Run keyword research prompt
- Generate initial outline
- Suggest structure and format
- Estimate word count

### Stage 2: Human Review & Expansion (HUMAN)
- Add first-hand experiences
- Insert proprietary data
- Include specific case studies
- Add expert interviews/quotes

### Stage 3: AI Draft Generation (AI)
- Generate full first draft using expanded outline
- Optimize for readability
- Add internal links
- Suggest metadata (title, description)

### Stage 4: Human Edit & Brand Voice (HUMAN)
- Enforce brand voice consistency
- Add opinions and contrarian takes
- Verify all claims and data
- Simplify or clarify complex sections

### Stage 5: AI Optimization (AI)
- Optimize for SEO (Surfer, MarketMuse)
- Generate meta descriptions
- Create social media variations
- Suggest related content links

### Stage 6: Final QA (HUMAN)
- Fact-check all claims
- Verify links work
- Check formatting
- Final brand review
- Publish
```
