# Webinar Delivery - Checklist

## Pre-Implementation

- [ ] Send reminder emails 24 hours before webinar
- [ ] Send reminder emails 1 hour before webinar
- [ ] Test all technical elements 15 minutes early
- [ ] Check audio, video, screen sharing quality
- [ ] Have backup plan for technical failures
- [ ] Start on time with clear welcome and agenda
- [ ] Introduce speakers with credentials
- [ ] Use engaging presentation techniques (polls, chat, questions)

## Implementation Phase

- [ ] Share screen clearly with adequate font sizes
- [ ] Pause regularly for Q&A and interaction
- [ ] Share relevant resources and links in chat
- [ ] Record webinar for replay access
- [ ] Announce offer/CTA clearly 2-3 times during webinar
- [ ] End with clear next steps and closing
- [ ] Thank attendees and confirm follow-up process
