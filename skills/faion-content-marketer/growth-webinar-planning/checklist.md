# Webinar Planning - Checklist

## Pre-Implementation

- [ ] Define webinar goals and success metrics
- [ ] Choose webinar topic and timing
- [ ] Research and book webinar platform (Zoom, Hopin, Demio)
- [ ] Create webinar marketing plan and promotion timeline
- [ ] Design landing page with registration form
- [ ] Create webinar email invitations (series of 3-5)
- [ ] Plan agenda and speaker roles
- [ ] Create presentation slides with speaker notes

## Implementation Phase

- [ ] Prepare live demo scripts and backup plans
- [ ] Set up automated emails for registrants (reminder, follow-up)
- [ ] Prepare Q&A responses for likely questions
- [ ] Test technical setup (audio, video, sharing)
- [ ] Create follow-up email sequence for attendees/non-attendees
- [ ] Plan offer/CTA for post-webinar conversion
- [ ] Set up metrics tracking (registrations, attendance, engagement)
