# TikTok Basics - Checklist

## Pre-Implementation

- [ ] Research TikTok trending sounds and effects
- [ ] Identify target audience demographics on TikTok
- [ ] Understand TikTok algorithm and recommendation system
- [ ] Plan content themes that resonate with TikTok users
- [ ] Invest in basic video recording setup (phone, lighting, audio)
- [ ] Create TikTok content calendar (minimum 3x/week)
- [ ] Use trending music and sounds in videos
- [ ] Keep videos concise (15-60 seconds optimal)

## Implementation Phase

- [ ] Add captions and text overlays for clarity
- [ ] Use trending hashtags relevant to content
- [ ] Engage with other creators' content (comments, shares)
- [ ] Post consistently to build algorithm familiarity
- [ ] Monitor analytics for top-performing content
- [ ] Identify trending formats and adapt them
- [ ] Plan for content evolution and audience feedback
