## Problem

TikTok has 1.5B+ monthly active users and the highest organic reach of any social platform. But most B2B founders and SaaS marketers dismiss it as "just for Gen Z dancing." Meanwhile, competitors are getting millions of views and acquiring customers at near-zero CAC.

The platform rewards creativity, authenticity, and trend awareness. Most businesses fail because they post corporate content that doesn't fit the platform's native style.

---

## Framework

TikTok growth follows a different pattern than other platforms:

```
HOOK      -> Capture attention in first 1-3 seconds
RETAIN    -> Keep viewers watching till the end
CONVERT   -> Turn viewers into followers/customers
COMPOUND  -> Let algorithm amplify your best content
```

## Content Strategy

### TikTok Content Pillars for B2B/SaaS

| Pillar | Description | Example |
|--------|-------------|---------|
| **Educational** | Quick tips, how-tos | "3 VS Code shortcuts you need" |
| **Behind the scenes** | Build in public | "Day 47 of building my startup" |
| **Trends with twist** | Use trends for your niche | Trending sound + product demo |
| **Storytelling** | Founder journey | "How I got my first 100 users" |
| **Myth busting** | Challenge assumptions | "Why your landing page sucks" |

### Video Formats That Work

| Format | Duration | Best For |
|--------|----------|----------|
| **Quick tips** | 15-30 sec | Educational content |
| **Story time** | 60-90 sec | Journey, lessons learned |
| **Tutorial** | 60-180 sec | How-to content |
| **POV/Skit** | 15-30 sec | Relatable humor |
| **Before/After** | 30-60 sec | Transformations, results |

## Templates

### Educational Video Script

```markdown
HOOK (0-3 sec):
"[Number] [thing] every [role] needs to know"
[Text on screen: same as spoken]

BODY (4-45 sec):
"Number one: [tip]"
[Show example/demonstration]

"Number two: [tip]"
[Show example/demonstration]

"Number three: [tip]"
[Show example/demonstration]

CTA (46-60 sec):
"Follow for more [topic] tips"
[Point to follow button]
```

### Build in Public Series

```markdown
HOOK:
"Day [X] of building [product]"
[Show face + quick context]

UPDATE:
"Today I [what you worked on]"
[Show screen recording or results]

LEARNING:
"What I learned: [insight]"
[Quick reflection]

ENGAGEMENT:
"What should I focus on tomorrow?"
[Ask for comments]
```

### Product Demo Video

```markdown
HOOK (0-3 sec):
"POV: You just discovered [product category] that actually works"

PROBLEM (4-10 sec):
"You know when [pain point]?"

SOLUTION (11-40 sec):
"[Product] lets you [solution]"
[Screen recording of product in action]

RESULT (41-50 sec):
"Now you can [benefit]"
[Show end result]

CTA (51-60 sec):
"Link in bio to try free"
```

## Implementation Checklist

### Week 1: Setup
- [ ] Create account (username, bio, profile photo)
- [ ] Research 10 successful accounts in your niche
- [ ] Define 3-4 content pillars
- [ ] Plan first 7 videos
- [ ] Set up basic filming setup (ring light, phone mount)

### Week 2-4: Post Daily
- [ ] Post 1 video every day
- [ ] Test different hooks
- [ ] Track which videos perform best
- [ ] Engage with comments within 1 hour
- [ ] Follow and interact with niche creators

### Month 2: Optimize
- [ ] Increase to 2-3 videos/day
- [ ] Double down on what works
- [ ] Start incorporating trends
- [ ] Create video series
- [ ] Optimize bio link for conversions

### Month 3+: Scale
- [ ] Aim for 3-5 videos/day
- [ ] Repurpose top content
- [ ] Test TikTok ads on best organic content
- [ ] Build relationships with other creators
- [ ] Start monetizing (leads, sales, partnerships)

## Common Mistakes

| Mistake | Why It Fails | Fix |
|---------|--------------|-----|
| Slow hooks | Viewers scroll past | Hook in first 1 second |
| Corporate style | Doesn't fit platform | Be authentic, casual |
| Inconsistent posting | Algorithm forgets you | Post daily minimum |
| Long intros | Kills watch time | Start mid-action |
| No CTA | Viewers don't convert | Clear CTA every video |
| Only promotional | No one follows ads | 90% value, 10% promo |
| Overproduced | Feels inauthentic | iPhone quality is fine |
