# Search Everywhere Optimization - Checklist

## Pre-Implementation

- [ ] Research search behavior across platforms (Google, YouTube, TikTok)
- [ ] Identify keywords for each search platform
- [ ] Optimize website for featured snippets
- [ ] Create YouTube video descriptions with keywords
- [ ] Optimize TikTok captions and hashtags
- [ ] Build backlink strategy for domain authority
- [ ] Create content optimized for people also ask sections
- [ ] Optimize for voice search (conversational keywords)

## Implementation Phase

- [ ] Create content for local search (if applicable)
- [ ] Optimize for image search (alt text, file names)
- [ ] Build mobile optimization for all platforms
- [ ] Create schema markup for rich snippets
- [ ] Set up monitoring for keyword rankings
- [ ] Plan periodic updates to content for freshness
- [ ] Build authority signals across platforms
