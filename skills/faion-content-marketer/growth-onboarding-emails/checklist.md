# Onboarding Emails - Checklist

## Pre-Implementation

- [ ] Map user journey and key activation points
- [ ] Define success metrics (feature adoption, time to first value)
- [ ] Identify critical onboarding emails (welcome, setup, first action, etc.)
- [ ] Write welcome email with clear next steps
- [ ] Create setup/activation email sequence
- [ ] Write feature education emails based on user segments
- [ ] Plan milestone celebration emails
- [ ] Create troubleshooting/help emails for common issues

## Implementation Phase

- [ ] Design email templates for mobile and desktop
- [ ] Set up triggered emails based on user behavior
- [ ] Add clear CTAs linking to in-app features or help docs
- [ ] Test email timing and frequency
- [ ] Monitor onboarding completion rates by email
- [ ] Create win/success email templates
- [ ] Plan upsell emails for power users
