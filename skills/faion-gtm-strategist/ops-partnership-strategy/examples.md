# Ops Partnership Strategy - Examples

### Example 1: SaaS Affiliate Partnership

**Partner:** Productivity blogger with 50K subscribers

**Proposal:**
- 30% recurring commission
- Exclusive discount code (20% off)
- Co-created review content

**Execution:**
1. Provided free account for testing
2. Created custom landing page
3. Supplied images and messaging
4. Monthly performance emails

**Results:**
- 25 signups in first month
- $750 in recurring commissions
- Featured in year-end tool roundup
