- [ ] Form LLC or corporation
- [ ] Get EIN

- [ ] Basic Terms of Service
- [ ] Basic Privacy Policy
- [ ] Cookie consent (if needed)

### Month 1 (After Launch)
- [ ] Review and improve policies

### Website Legal Requirements

```markdown
## Website Legal Pages

### Required Pages
1. **Terms of Service**
   - Link in footer
   - Checkbox at signup

2. **Privacy Policy**
   - Link in footer
   - Link in signup flow

3. **Cookie Policy** (if using cookies)
   - Banner with consent
   - Link to full policy

### Recommended Additions
4. **Refund Policy**
   - Clear conditions
   - Process explained

5. **Disclaimer**
   - Earnings disclaimers (if applicable)
   - Professional advice disclaimers

### Footer Example
[Terms] | [Privacy] | [Cookies] | [Refund Policy]
```

### Policy Update Process

```markdown
## Policy Update Procedure

### When to Update
- New features collecting data
- New third-party integrations
- Regulatory changes
- Business model changes

### Update Steps
1. Draft updated policy
2. Legal review (if significant)
3. Note changes (changelog)
4. Notify users (email for material changes)
5. Update effective date
6. Publish new version

### User Notification (Material Changes)
Subject: Updates to our [Terms/Privacy Policy]

We've updated our [policy] to [brief reason].

Key changes:
- [Change 1]
- [Change 2]

The updated policy takes effect on [date].

[Read full policy]
```

## Common Mistakes

| Mistake | Why It Fails | Fix |
|---------|--------------|-----|
| No LLC | Personal liability | Form an LLC |
| Copy-paste policies | May not fit your business | Customize or use service |
| No privacy policy | Legally required | Create one before launch |
| Ignoring GDPR | Fines, blocked in EU | Comply from start |
| No contracts | Disputes, unclear terms | Use written agreements |
| Outdated policies | Don't match business | Review annually |
