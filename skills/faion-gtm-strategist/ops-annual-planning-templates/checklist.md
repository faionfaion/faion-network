# Annual Planning Templates Checklist

## Pre-Planning Phase

### Stakeholder Alignment
- [ ] Executive team aligned on vision
- [ ] Finance approval for budget envelope
- [ ] Board/investors informed (if applicable)
- [ ] Key team leads identified for input

### Data Gathering
- [ ] Previous year performance reviewed
- [ ] Market/competitive analysis available
- [ ] Team feedback and retrospectives collected
- [ ] Customer feedback and requests compiled

---

## Annual Plan Creation

### Planning Schedule
- [ ] Start date set (typically Aug-Sep for Jan launch)
- [ ] Stakeholder input deadline defined
- [ ] Draft review date scheduled
- [ ] Approval/sign-off date confirmed
- [ ] Communication plan prepared

### Vision & Theme Development
- [ ] Company vision statement reviewed/updated
- [ ] Annual theme selected and defined
- [ ] Key strategic priorities identified (3-5 maximum)
- [ ] Vision communicated to team

### Goal Setting
- [ ] SMART goals defined (Specific, Measurable, Achievable, Relevant, Time-bound)
- [ ] Goals aligned with theme and priorities
- [ ] Success metrics and targets set
- [ ] Historical baseline data gathered for comparison

### Quarterly Breakdown
- [ ] Q1 goals defined (aligned with annual)
- [ ] Q2 goals defined
- [ ] Q3 goals defined
- [ ] Q4 goals defined
- [ ] Dependencies between quarters identified
- [ ] Key initiatives per quarter listed

### Budget Planning
- [ ] Marketing budget allocated
- [ ] Tools/software budget estimated
- [ ] Contractor/freelance budget included
- [ ] Education/conference budget set
- [ ] Contingency budget (10-15%) reserved
- [ ] Monthly burn rate calculated

### Resource Planning
- [ ] Team capacity assessed
- [ ] Hiring needs identified (if any)
- [ ] Contractor/outsourcing needs defined
- [ ] Skills gaps identified for training

### Risk Planning
- [ ] Key risks identified
- [ ] Mitigation strategies for each risk
- [ ] Contingency plans documented
- [ ] Assumptions documented

### Review Cadence
- [ ] Weekly review day/time set
- [ ] Monthly review format defined
- [ ] Quarterly review schedule confirmed
- [ ] Annual review date set
- [ ] Review participants identified

---

## Quarterly Review Process

### Before Review
- [ ] Goals vs. actual progress compiled
- [ ] Metrics dashboard updated
- [ ] Team feedback solicited
- [ ] Win/miss analysis prepared

### During Review
- [ ] Results vs. goals presented
- [ ] Wins celebrated
- [ ] Misses analyzed (why, lessons learned)
- [ ] Blockers/risks discussed
- [ ] Next quarter priorities reviewed

### Post-Review
- [ ] Decisions documented
- [ ] Action items assigned (with owners)
- [ ] Plan adjustments made if needed
- [ ] Team communication completed
- [ ] Updated plan documented

---

## Budget Tracking

### Monthly
- [ ] Spend vs. budget reviewed
- [ ] Burn rate analyzed
- [ ] Adjustments made if needed
- [ ] Forecast for rest of year updated

### Quarterly
- [ ] Full budget review completed
- [ ] Category-by-category analysis done
- [ ] ROI on major expenses calculated
- [ ] Reallocation approved if needed

### Annual
- [ ] Full year budget vs. actual reviewed
- [ ] ROI analysis for each category
- [ ] Lessons for next year documented
- [ ] Next year budget informed by results

---

## Goals & Metrics Tracking

### Baseline Establishment
- [ ] Current state metrics measured
- [ ] Target (end of year) defined per goal
- [ ] Stretch goals identified (if applicable)

### Weekly Tracking
- [ ] Progress checked against milestones
- [ ] Early warning indicators monitored
- [ ] Course corrections made as needed

### Monthly Reporting
- [ ] Metric values recorded
- [ ] Trend analysis performed
- [ ] Insights shared with team
- [ ] Motivation maintained

### Quarterly Assessment
- [ ] Quarter progress toward annual goal reviewed
- [ ] Pacing (on track/behind/ahead) assessed
- [ ] Adjustments made to strategy if needed
- [ ] Forecasting for final quarters updated

---

## Documentation

### Planning Documents
- [ ] Annual plan document created
- [ ] OKRs (if using that framework) documented
- [ ] Supporting strategic priorities explained
- [ ] Key initiatives listed
- [ ] Budget breakdown detailed
- [ ] Timeline/milestones specified

### Team Communication
- [ ] Plans shared with entire team
- [ ] Q&A session conducted
- [ ] Commitment from team obtained
- [ ] Individual/team goals aligned with plan

### Progress Artifacts
- [ ] Weekly status updates maintained
- [ ] Monthly review summaries documented
- [ ] Quarterly deep-dive analysis completed
- [ ] Metrics dashboard kept updated

### Learnings
- [ ] What worked documented for next year
- [ ] What didn't work analyzed
- [ ] Assumptions that proved wrong noted
- [ ] Surprises captured

---

## Annual Review

### Results Analysis
- [ ] All annual goals reviewed
- [ ] Achievement rates calculated
- [ ] Performance vs. benchmarks analyzed
- [ ] Year-over-year trend analyzed

### Learnings & Insights
- [ ] Top 3-5 learnings identified
- [ ] Strategic insights captured
- [ ] Team feedback collected
- [ ] Customer insights integrated

### Story Development
- [ ] Year narrative prepared
- [ ] Major wins highlighted
- [ ] Challenges overcome explained
- [ ] Impact to business/customers shown

### Planning for Next Year
- [ ] Momentum identified for next year
- [ ] Gaps/misses addressed in new plan
- [ ] Emerging opportunities captured
- [ ] Strategic direction adjusted if needed

---

## Tools & Systems

### Planning Tools
- [ ] Spreadsheet/database selected (Notion, Google Sheets, etc.)
- [ ] Access controls configured
- [ ] Templates created in tool
- [ ] Notification/reminder system set up

### Communication
- [ ] Team communication channel for plan
- [ ] Review meeting calendar invites sent
- [ ] Status update format defined
- [ ] Reporting cadence set

### Tracking
- [ ] Metrics dashboard created
- [ ] Automated data feeds configured (if possible)
- [ ] Manual data entry process defined
- [ ] Visualization/reporting prepared
