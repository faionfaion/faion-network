## Platform-Specific Strategy

### YouTube

**Specs:**
- Resolution: 1920x1080 or 4K
- Aspect: 16:9
- Duration: 7-15min optimal for retention
- Format: MP4, H.264 codec

**Optimization:**
- First 30s critical for retention
- Add end screen (last 20s)
- Include chapter markers
- Optimize title, description, tags for SEO
- Custom thumbnail (1280x720)

**Production:**
- Use Runway Gen-4 for high-quality segments
- Combine with screen recording for tutorials
- Add subtitles (burned-in or SRT file)

### TikTok

**Specs:**
- Resolution: 1080x1920
- Aspect: 9:16 (vertical)
- Duration: 15-60s (peak engagement at 21-34s)
- Format: MP4

**Optimization:**
- Hook in first 3 seconds
- Fast-paced cuts (every 2-3s)
- On-screen text for no-sound viewing
- Trending audio increases reach
- Call-to-action at end

**Production:**
- Use Pika Labs for speed and effects
- Pikaffects for viral potential
- Generate vertical from start (don't crop horizontal)

### Instagram Reels

**Specs:**
- Resolution: 1080x1920
- Aspect: 9:16
- Duration: 15-90s
- Format: MP4

**Optimization:**
- First frame should be engaging (no black)
- Use trending audio
- Include captions
- End with strong CTA

**Production:**
- Similar to TikTok
- Can cross-post but optimize separately
- Use Instagram stickers and features

### LinkedIn

**Specs:**
- Resolution: 1920x1080
- Aspect: 16:9
- Duration: 30s-3min
- Format: MP4

**Optimization:**
- Professional tone
- Educational or thought leadership content
- Subtitles essential (85% watch without sound)
- Native upload performs better than YouTube links

**Production:**
- Use Sora 2 for professional quality
- Talking head + B-roll format
- Lower third graphics for credibility

### Twitter/X

**Specs:**
- Resolution: 1280x720
- Aspect: 16:9
- Duration: Max 2:20
- File size: Under 512MB

**Optimization:**
- Hook in first 2 seconds
- Keep under 45 seconds for auto-play
- Text overlay critical
- Engagement happens in first hour

**Production:**
- Use Pika Labs for quick turnaround
- Text-heavy content works well
- GIF format for loops (under 5s)

## Distribution Strategy

### Multi-Platform Workflow

```
1. MASTER CONTENT
   ├── Create horizontal (16:9) master
   ├── Export at highest quality
   └── Archive source files

2. PLATFORM ADAPTATIONS
   ├── Crop to 9:16 for TikTok/IG Reels
   ├── Create 1:1 square for Instagram feed
   ├── Add platform-specific elements
   └── Optimize titles/descriptions per platform

3. SCHEDULING
   ├── YouTube: Tuesday/Thursday 2PM EST
   ├── TikTok: Daily 9AM, 12PM, 7PM EST
   ├── Instagram: Daily 11AM, 7PM EST
   ├── LinkedIn: Tuesday-Thursday 8AM EST
   └── Twitter: Multiple times daily

4. ANALYTICS
   ├── Track performance by platform
   ├── Identify best-performing content
   ├── Iterate on successful formats
   └── A/B test thumbnails, hooks, CTAs
```

### Cross-Platform Repurposing

```
Long-form YouTube (10min)
  ↓
├── Extract 3-5 key moments (60s each)
│   ↓
│   ├── Crop to 9:16 → Instagram Reels
│   ├── Add trending audio → TikTok
│   └── Add professional context → LinkedIn
│
├── Create teaser (30s)
│   └── Share on Twitter with YouTube link
│
└── Extract quotes
    └── Text posts across all platforms
```

## Content Hooks and Frameworks

### First 3 Seconds Hooks

| Hook Type | Example |
|-----------|---------|
| **Question** | "Want to 10x your conversions?" |
| **Stat** | "93% of marketers fail at this..." |
| **Promise** | "Here's how to build this in 5 minutes" |
| **Curiosity** | "The secret nobody talks about..." |
| **Pattern Interrupt** | [Unexpected visual or sound] |

### Video Frameworks

**AIDA (Awareness, Interest, Desire, Action):**
```
0-3s:  Attention-grabbing hook
3-15s: Build interest with problem/solution
15-45s: Create desire with benefits/proof
45-60s: Clear call-to-action
```

**PAS (Problem, Agitate, Solution):**
```
0-10s:  State the problem
10-30s: Agitate (make it personal/urgent)
30-60s: Present your solution + CTA
```

**Story Arc:**
```
0-5s:   Hook/setup
5-20s:  Challenge/conflict
20-45s: Resolution
45-60s: Lesson + CTA
```

## Video SEO

### YouTube SEO Checklist

- [ ] Keyword in title (front-loaded)
- [ ] Keyword in first 25 words of description
- [ ] 3-5 relevant tags
- [ ] Custom thumbnail with text
- [ ] Closed captions (auto + manual review)
- [ ] Chapter markers
- [ ] Cards and end screens
- [ ] Playlist organization
- [ ] Pin comment with CTA
- [ ] Link to related videos

### Thumbnail Best Practices

| Element | Guideline |
|---------|-----------|
| **Text** | Max 3-4 words, 40pt+ font |
| **Faces** | High emotion, eye contact |
| **Contrast** | Bright colors, high contrast |
| **Branding** | Consistent style across videos |
| **Test** | A/B test different thumbnails |

## Paid Video Advertising

### Meta Ads (Facebook/Instagram)

**Video Ad Specs:**
- Feed: 1:1 (1080x1080) or 4:5 (1080x1350)
- Stories: 9:16 (1080x1920)
- Reels: 9:16 (1080x1920)
- Duration: 15s optimal, max 240s

**Best Practices:**
- Hook in first 3 seconds
- Show product in first 5 seconds
- Include captions (sound often off)
- Test multiple variations
- Use video views for warm audiences

### Google Video Ads (YouTube)

**Ad Types:**
- Skippable In-Stream (5s before skip)
- Non-Skippable (15-20s)
- Bumper (6s)
- Discovery (in search/watch next)

**Best Practices:**
- Front-load brand/product
- Optimize for first 5 seconds
- Include CTA overlay
- Target by audience + content
- Use remarketing for conversions

### TikTok Ads

**Specs:**
- Resolution: 720x1280 or 1080x1920
- Duration: 5-60s (9-15s optimal)
- File size: <500MB

**Best Practices:**
- Look native (not polished/corporate)
- Use TikTok-native sounds
- Fast-paced editing
- Influencer-style delivery
- Test spark ads (boost organic content)
