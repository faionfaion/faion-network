## Key Concepts

### Campaign Structure

```
Meta:   Campaign (Objective) > Ad Set (Audience) > Ad (Creative)
Google: Campaign (Type) > Ad Group (Keywords) > Ad (Responsive)
LinkedIn: Campaign (Objective) > Ad (Sponsored Content/Lead Gen)
```

### Key Metrics

| Metric | Formula | Good Benchmark |
|--------|---------|----------------|
| CPM | Cost / Impressions x 1000 | $5-20 |
| CPC | Cost / Clicks | $0.50-2.00 |
| CTR | Clicks / Impressions x 100 | 1%+ |
| CVR | Conversions / Clicks x 100 | 2%+ |
| CPA | Cost / Conversions | Varies by industry |
| ROAS | Revenue / Ad Spend | 3:1+ |

### Audience Types

| Type | Platform | Best For |
|------|----------|----------|
| Interest-based | Meta, Google Display | Cold traffic, awareness |
| Keyword-based | Google Search | High-intent acquisition |
| Custom/Remarketing | All | Warm traffic, retargeting |
| Lookalike/Similar | Meta, Google | Scaling proven audiences |
| Professional | LinkedIn | B2B decision-makers |
