## Key Frameworks

### AIDA (Landing Pages, Copy)

```
Attention -> Interest -> Desire -> Action
```

### PAS (Problem-focused Copy)

```
Problem -> Agitate -> Solution
```

### Launch Channels Priority

| Channel | Type | Best For |
|---------|------|----------|
| Product Hunt | Launch | B2B SaaS, developer tools |
| Hacker News | Launch | Technical products, open source |
| Twitter/X | Ongoing | Tech audience, building in public |
| LinkedIn | Ongoing | B2B, professional services |
| Reddit | Ongoing | Niche communities |
| Newsletter | Ongoing | Owned audience |
