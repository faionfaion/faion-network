# LLM Prompts

## Analysis Prompts

### Prompt 1: Initial Assessment

```
Act as an expert project manager. Analyze the following [context/situation]:

[Input placeholder]

Provide:
1. Key observations
2. Potential risks
3. Recommended approach
4. Success criteria
```

---

### Prompt 2: Gap Analysis

```
Review the current state vs. desired state:

**Current State:** [Description]
**Desired State:** [Description]

Identify:
1. Critical gaps
2. Dependencies
3. Prioritized action items
4. Resource requirements
```

---

## Generation Prompts

### Prompt 3: Document Generation

```
Create a [document type] for [project context]:

**Requirements:**
- Requirement 1
- Requirement 2

**Constraints:**
- Constraint 1
- Constraint 2

Format: [Specify format]
Tone: [Professional/Technical/Executive]
```

---

### Prompt 4: Problem Solving

```
As a project management expert, help solve this challenge:

**Problem:** [Description]
**Constraints:** [List constraints]
**Stakeholders:** [List stakeholders]
**Timeline:** [Timeline]

Provide:
1. Root cause analysis
2. Solution options (with pros/cons)
3. Recommended approach
4. Implementation steps
```

---

## Review Prompts

### Prompt 5: Quality Check

```
Review the following [deliverable] for quality and completeness:

[Paste content]

Check for:
1. Completeness (all sections present)
2. Consistency (terminology, format)
3. Clarity (understandable to audience)
4. Actionability (clear next steps)

Provide specific improvement suggestions.
```

---

## Coaching Prompts

### Prompt 6: Best Practices

```
What are the current best practices for [methodology aspect] in [year/context]?

Focus on:
1. Industry standards
2. Tool recommendations
3. Common pitfalls
4. Success patterns
```
