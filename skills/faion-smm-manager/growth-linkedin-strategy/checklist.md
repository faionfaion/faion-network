## Phase 2: Content Strategy

### Content Pillars for LinkedIn
- [ ] Pillar 1: Professional insights/industry trends
- [ ] Pillar 2: Thought leadership/expertise
- [ ] Pillar 3: Behind-the-scenes/company culture
- [ ] Pillar 4: Helpful tips and resources
- [ ] Pillar 5: Personal/authentic stories
- [ ] Document allocation percentage
- [ ] Create content examples for each

### Content Types
- [ ] Articles (long-form, 1500+ words)
- [ ] Posts (mixed format, text/images)
- [ ] Videos (6-60 seconds)
- [ ] Document shares (PDF, presentation)
- [ ] Carousels (10-20 images with captions)
- [ ] Native videos (highest engagement)
- [ ] Polls (engagement and insights)

### LinkedIn Best Practices
- [ ] Posts with images get 2x more engagement
- [ ] Videos get 5x more engagement
- [ ] First 2 lines are critical (appear before "see more")
- [ ] Use line breaks for readability
- [ ] Include calls-to-action
- [ ] Post 2-3 times per week
- [ ] Optimal timing: Tuesday-Thursday, 8am-10am

## Phase 3: Content Creation

### Content Calendar
- [ ] Create 30-day content calendar
- [ ] Plan mix of content types
- [ ] Schedule post times
- [ ] Identify trending topics
- [ ] Plan industry commentary
- [ ] Plan personal stories
- [ ] Document content themes

### Writing for LinkedIn
- [ ] Start with attention-grabbing hook
- [ ] Keep paragraphs short (2-3 lines max)
- [ ] Use line breaks between sections
- [ ] Include specific numbers and data
- [ ] Use 2-3 hashtags (strategically)
- [ ] Include clear call-to-action
- [ ] Authentic and conversational tone

### Content Creation Schedule
- [ ] Batch create monthly content
- [ ] Create 20-30 posts per month minimum
- [ ] Create 2-3 long-form articles per month
- [ ] Record 2-3 videos per month
- [ ] Maintain content library
- [ ] Use scheduling feature
- [ ] Review before publishing
