# Instagram Basics Checklist

## Phase 1: Profile Setup

### Account Creation & Optimization
- [ ] Create business/creator account (not personal)
- [ ] Professional profile photo (headshot or logo)
- [ ] Compelling bio (150 chars max)
- [ ] Include keywords and value prop
- [ ] Add website/link in bio
- [ ] Write "call to action" in bio (link in bio, follow, etc.)
- [ ] Add contact info (email, phone, location)
- [ ] Switch to Creator Account for insights

### Profile Branding
- [ ] Define visual aesthetic/brand colors
- [ ] Create consistent image style
- [ ] Plan feed aesthetic layout
- [ ] Create branded templates (Canva)
- [ ] Pin top 3 posts to profile
- [ ] Create highlights from stories (for pinning)
- [ ] Document brand guidelines

## Phase 2: Content Types & Strategy

### Instagram Content Formats
- [ ] Feed posts (carousel or single image, 3:4 ratio)
- [ ] Stories (24-hour content, 9:16 ratio)
- [ ] Reels (video 15-60 seconds, high engagement)
- [ ] Live video (real-time engagement)
- [ ] IGTV/long-form video
- [ ] Guides (collections of posts/places)
- [ ] Direct messaging

### Content Pillars
- [ ] Pillar 1: Showcase products/services (40%)
- [ ] Pillar 2: Educational tips/how-tos (30%)
- [ ] Pillar 3: Behind-the-scenes/lifestyle (20%)
- [ ] Pillar 4: Community/engagement (10%)
- [ ] Document content mix percentages

### Instagram Best Practices
- [ ] Posts with Reels get 67% higher engagement
- [ ] Post 3-5 times per week (consistency matters)
- [ ] 5-10 stories per week minimum
- [ ] 1-2 Reels per week (increasing importance)
- [ ] Optimal posting time: morning 6-9am, evening 7-9pm
- [ ] Captions should be personal, not corporate

## Phase 3: Content Creation

### Caption Strategy
- [ ] Hook in first line (emoji or question)
- [ ] Keep under 2200 characters (sweet spot: 150-200)
- [ ] Tell a story, not just product pitch
- [ ] Use 3-5 relevant hashtags (at end or comment)
- [ ] Include call-to-action
- [ ] Use line breaks for readability
- [ ] Be authentic and conversational

### Visual Content
- [ ] Feed posts: Consistent color/aesthetic
- [ ] Create image templates (Canva)
- [ ] Use high-quality images (no blurry/pixelated)
- [ ] Include text overlays for clarity
- [ ] Mobile-optimized (test readability)
- [ ] Brand consistency across posts
- [ ] Update visual style quarterly

### Story Strategy
- [ ] Post 3-5 stories daily
- [ ] Show behind-the-scenes content
- [ ] Create polls/questions for engagement
- [ ] Use story features (stickers, text, filters)
- [ ] Create swipe-up links (if eligible)
- [ ] Save best stories as highlights
- [ ] Use countdown stickers for launches

### Reels Strategy (Critical for growth)
- [ ] Trend-aware content
- [ ] 15-30 seconds optimal length
- [ ] Hook in first 1 second
- [ ] Trending sounds (important)
- [ ] Text overlays for engagement
- [ ] 1-2 Reels per week minimum
- [ ] Post during peak engagement times

## Phase 4: Hashtag Strategy

### Hashtag Selection
- [ ] Identify 20-30 relevant hashtags
- [ ] Mix: Large (1M+), Medium (100k-1M), Small (10k-100k)
- [ ] Avoid banned/overused hashtags
- [ ] Research trending hashtags in niche
- [ ] Create branded hashtag
- [ ] Monitor hashtag performance
- [ ] Rotate hashtags to avoid spam flagging

### Hashtag Implementation
- [ ] Use 3-5 hashtags (more looks spammy)
- [ ] Place at end of caption or first comment
- [ ] Include branded hashtag
- [ ] Test different hashtag combinations
- [ ] Track which hashtags drive engagement
- [ ] Update hashtags based on performance
- [ ] Stay relevant to content

## Phase 5: Engagement & Community

### Daily Engagement Activities
- [ ] Spend 15-20 minutes engaging daily
- [ ] Engage with 20-30 accounts daily
- [ ] Reply to all comments within 24 hours
- [ ] Like and comment on audience posts
- [ ] Tag relevant accounts in posts
- [ ] Share posts to stories (reposting)
- [ ] Build genuine community

### Network Building
- [ ] Follow 20-50 relevant accounts per week
- [ ] Engage with micro-influencers (10k-100k followers)
- [ ] Comment meaningfully on their posts
- [ ] Build relationships before asking for anything
- [ ] Avoid aggressive follow/unfollow tactics
- [ ] Focus on quality engagement over quantity
- [ ] Document relationship building progress

### DM Strategy
- [ ] Respond to DMs within 24 hours
- [ ] Personalized DM replies
- [ ] Use DMs for customer service
- [ ] Build relationships via DM
- [ ] Create DM sequences for leads
- [ ] Use auto-reply for after hours
- [ ] Track DM conversions
