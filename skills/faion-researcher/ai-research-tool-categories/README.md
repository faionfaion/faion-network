# AI Research Tool Categories

## Quick Reference

### By Research Phase

| Phase | Tools |
|-------|-------|
| **Exploration** | Perplexity, Claude, ChatGPT |
| **Competitor Intel** | Crayon, Klue, Contify, AlphaSense |
| **User Interviews** | Looppanel, Dovetail, User Interviews |
| **Survey Analysis** | Qualtrics, SurveyMonkey AI |
| **Sentiment** | Brandwatch, Sprout Social |
| **Synthesis** | Miro AI, Condens, EnjoyHQ |
| **Synthetic Research** | Synthetic Users, Viewpoints.ai |

### By Budget

| Budget | Stack |
|--------|-------|
| Free/Low | Perplexity, Claude, Google Trends |
| Mid | Looppanel, Dovetail, basic Crayon |
| Enterprise | AlphaSense, Klue, full Crayon, Userlytics |
