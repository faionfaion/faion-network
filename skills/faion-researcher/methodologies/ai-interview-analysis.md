## AI-Assisted Interview Analysis

### Problem

Manual interview transcription and analysis is time-consuming.

### Solution: AI-Powered Interview Tools

**Interview Analysis Stack:**

| Tool | Best For |
|------|----------|
| Looppanel | UX research, 17 languages, 90%+ accuracy |
| Dovetail | Research repository, pattern detection |
| Insight7 | Bulk interview analysis, sentiment |
| UserBit | 50+ languages, speaker identification |
| User Interviews | Session breakdown by discussion guide |
| Otter.ai | Real-time transcription |
| Fireflies.ai | Meeting transcription |
| Descript | Video editing + transcription |

**AI Interview Analysis Capabilities:**
```
Recording -> Transcription -> Speaker ID -> Summarization
                                              |
                               Theme Extraction
                                              |
                               Sentiment Analysis
                                              |
                               Pattern Detection
                                              |
                               Insight Generation
```

**Time Savings:**
- Transcription: Hours -> Minutes
- Summarization: Hours -> Seconds
- Theme extraction: Days -> Hours
- Cross-interview patterns: Weeks -> Hours

**Accuracy Benchmarks (2025-2026):**
- State-of-the-art transcription: Up to 99% on good audio
- Speaker identification: Good reliability, occasional misidentification
- Sentiment analysis: Improving but requires human review

**Key Limitations:**
- Cannot observe/facilitate moderated studies
- Cannot understand nonverbal interactions
- Text-focused, limited behavioral analysis
- Requires human interpretation of insights
