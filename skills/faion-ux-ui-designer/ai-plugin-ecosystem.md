### Best Practices for AI Plugins

```
DO:
→ Use for repetitive tasks
→ Validate AI outputs
→ Combine multiple tools
→ Set up brand guidelines first

DON'T:
→ Accept AI output blindly
→ Skip accessibility review
→ Ignore design system constraints
→ Over-rely on generation
```
