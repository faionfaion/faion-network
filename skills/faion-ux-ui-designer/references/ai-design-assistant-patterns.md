# AI Design Assistant Patterns

### Problem

Designers need context-aware help without breaking flow.

### Solution: Integrated AI Assistants

**Assistant Types:**

| Type | Function | Example |
|------|----------|---------|
| Contextual | Suggestions based on current work | Figma AI suggestions |
| Generative | Create new elements on demand | Make, Draw tools |
| Review | Audit and feedback | Accessibility checkers |
| Documentation | Auto-generate specs | Design system docs |

**Interaction Patterns:**
```
Sidebar Assistant:
- Always visible
- Context-aware suggestions
- Quick actions

Modal Assistant:
- Focused generation tasks
- Complex operations
- Batch processing

Inline Assistant:
- In-place suggestions
- Quick edits
- Micro-improvements
```

**Human-AI Collaboration:**
```
AI excels at:
- Pattern recognition
- Variation generation
- Repetitive tasks
- Data processing

Humans excel at:
- Creative direction
- Emotional understanding
- Brand judgment
- Strategic decisions
```
