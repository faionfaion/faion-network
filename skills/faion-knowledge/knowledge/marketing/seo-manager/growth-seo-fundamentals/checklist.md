# SEO Fundamentals Checklist

## Phase 1: Foundation & Setup

### Technical Audit
- [ ] Set up Google Search Console
- [ ] Verify domain ownership
- [ ] Set preferred domain (www vs non-www)
- [ ] Submit XML sitemap
- [ ] Set crawl rate preferences
- [ ] Check robots.txt in Search Console

### Analytics Setup
- [ ] Install Google Analytics 4
- [ ] Create GA4 property
- [ ] Set up search console linking
- [ ] Configure conversion tracking
- [ ] Create custom reports for organic traffic

### Initial Technical Audit
- [ ] Test site indexing: `site:domain.com`
- [ ] Check Core Web Vitals in PageSpeed Insights
- [ ] Verify HTTPS certificate
- [ ] Test mobile responsiveness
- [ ] Check canonical tags on critical pages
- [ ] Audit robots.txt for blocking issues

## Phase 2: On-Page SEO Optimization

### Keyword Research (50+ keywords)
- [ ] Identify 50+ long-tail keywords (intent-based)
- [ ] Analyze keyword difficulty (tools: Ahrefs, Semrush, Moz)
- [ ] Segment by intent: informational, commercial, transactional
- [ ] Create keyword map spreadsheet
- [ ] Identify target keyword for each existing page

### Optimize Existing Pages
- [ ] Audit top 10 pages for on-page optimization
- [ ] Update title tags (primary keyword + benefit, <60 chars)
- [ ] Write meta descriptions (compelling, 120-160 chars)
- [ ] Ensure single H1 per page with keyword
- [ ] Create logical H2-H6 hierarchy
- [ ] Add internal links to related content
- [ ] Optimize images: alt text, compression, file names
- [ ] Add external links to authoritative sources
- [ ] Update "last updated" dates on refreshed content

### Content Depth Analysis
- [ ] Analyze top 3 SERP results for each target keyword
- [ ] Document word counts, headers, sections
- [ ] Identify gaps your content can fill
- [ ] Ensure content is comprehensive and complete
- [ ] Add missing sections before competitor content
- [ ] Include related entities and subtopics

## Phase 3: Technical SEO Implementation

### Site Structure
- [ ] Review and optimize URL structure
- [ ] Implement canonical tags (self-referencing)
- [ ] Set up proper redirect chains (301 redirects)
- [ ] Create logical site hierarchy (max 3 levels)
- [ ] Fix any orphan pages

### Core Web Vitals Optimization
- [ ] Optimize LCP (Largest Contentful Paint): <2.5s
  - [ ] Preload critical resources
  - [ ] Use WebP/AVIF image formats
  - [ ] Implement lazy loading below-fold
  - [ ] Optimize server response time
- [ ] Optimize INP (Interaction to Next Paint): <200ms
  - [ ] Minimize JavaScript blocking
  - [ ] Break tasks into <50ms chunks
  - [ ] Defer non-critical JS
- [ ] Optimize CLS (Cumulative Layout Shift): <0.1
  - [ ] Add width/height to images
  - [ ] Reserve space for ads and embeds
  - [ ] Use `font-display: swap`

### Structured Data & Schema
- [ ] Implement basic schema.org markup
- [ ] Add Organization schema (company info)
- [ ] Add Article schema (blog posts)
- [ ] Add BreadcrumbList schema (navigation)
- [ ] Test with Rich Results Test tool
- [ ] Validate schema.org syntax

## Phase 4: Content Calendar & Planning

### Content Strategy
- [ ] Create content calendar (12-month plan)
- [ ] Map keywords to content pieces (1 primary, 1-2 supporting)
- [ ] Plan 1-2 content pieces/week minimum
- [ ] Prioritize long-tail keywords (easier to rank)
- [ ] Plan cluster topics (pillar + sub-pages)

### Internal Linking Strategy
- [ ] Map topic clusters (main topic + subtopics)
- [ ] Create pillar content (comprehensive guide)
- [ ] Create cluster pages (detailed subtopics)
- [ ] Link pillar → cluster pages
- [ ] Link cluster pages → pillar
- [ ] Add contextual links in blog content
- [ ] Update old posts to link to new content

## Phase 5: Link Building & Off-Page

### Backlink Analysis
- [ ] Analyze competitor backlinks (Ahrefs, Moz)
- [ ] Identify high-authority linking sites (DA 40+)
- [ ] List sites linking to competitors but not you
- [ ] Identify broken links opportunities
- [ ] Find resource page link opportunities

### Link Building Campaign
- [ ] Create link-worthy content asset (guide, tool, research)
- [ ] Build prospect list (100+ sites)
- [ ] Research relevant contexts for link placement
- [ ] Find contact information (Hunter, Apollo)
- [ ] Personalize outreach emails
- [ ] Send initial outreach (20-50/day)
- [ ] Follow up after 7 days (once only)
- [ ] Track responses and conversions

### HARO & Press Coverage
- [ ] Sign up for HARO (Help A Reporter Out)
- [ ] Monitor journalist requests in your niche
- [ ] Respond to relevant queries with expert perspective
- [ ] Track featured mentions

## Phase 6: Monitoring & Reporting

### Monthly Reporting
- [ ] Track organic traffic (Google Analytics)
- [ ] Monitor keyword rankings (top 100)
- [ ] Count indexed pages (site:domain.com)
- [ ] Measure backlinks gained (new referring domains)
- [ ] Calculate ranking changes
- [ ] Review Core Web Vitals status
- [ ] Document learnings and wins

### KPI Tracking
- [ ] Organic traffic month-over-month growth
- [ ] Keywords in top 10 (target: +5-10/month)
- [ ] Domain authority trend
- [ ] Backlink growth (quality, not quantity)
- [ ] Click-through rate from SERPs
- [ ] Bounce rate from organic traffic

### Optimization & Iteration
- [ ] Identify top performers and replicate success
- [ ] Update underperforming content
- [ ] Double down on keywords in positions 11-20 (easy wins)
- [ ] Analyze search intent mismatches
- [ ] Refine content strategy based on data
- [ ] Test new content formats
