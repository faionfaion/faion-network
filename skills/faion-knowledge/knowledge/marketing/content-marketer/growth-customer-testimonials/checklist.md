# Customer Testimonials & Social Proof - Checklist

## Pre-Implementation

- [ ] Identify 10+ happiest customers for testimonials
- [ ] Determine optimal moments to request testimonials (post-win, post-milestone)
- [ ] Craft testimonial request emails with specific examples
- [ ] Ask targeted questions (before/after, specific results, surprise elements)
- [ ] Create video testimonial request sequence
- [ ] Follow up on non-responses (3-email sequence)
- [ ] Convert vague quotes to specific ones with numbers
- [ ] Request customer photos and company information

## Implementation Phase

- [ ] Get approvals on final testimonial versions
- [ ] Create case study template and research process
- [ ] Place testimonials strategically (above fold, near CTAs, pricing page)
- [ ] Add testimonials to landing pages with photos and titles
- [ ] Test testimonial impact on conversion rates
- [ ] Set up post-purchase automation for testimonial requests
- [ ] Create quarterly testimonial refresh process
