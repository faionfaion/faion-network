# Checklist - Operational Dashboard Setup

## Planning Phase

- [ ] Identify dashboard users (exec, department, team)
- [ ] Define dashboard purpose (monitoring, reporting, decisions)
- [ ] List all metrics needed
- [ ] Prioritize metrics by importance
- [ ] Determine update frequency
- [ ] Choose dashboard tool

## Tool Selection

- [ ] Evaluate dashboard tools (Tableau, Looker, Metabase, etc.)
- [ ] Choose based on data sources available
- [ ] Verify security and compliance
- [ ] Test performance with data volume
- [ ] Confirm integration capabilities
- [ ] Set up dev/staging/prod environments

## Data Foundation

- [ ] Identify all data sources
- [ ] Set up data warehouse or database
- [ ] Configure data pipelines
- [ ] Ensure data accuracy and freshness
- [ ] Set data refresh schedule
- [ ] Document data lineage

## Metric Selection

- [ ] Define AARRR metrics for dashboard
- [ ] Select North Star Metric prominently
- [ ] Add operational metrics
- [ ] Choose secondary KPIs
- [ ] Define targets for each metric
- [ ] Set benchmark comparisons

## Dashboard Design

- [ ] Design executive dashboard (3-5 metrics)
- [ ] Create department dashboards
- [ ] Design team-level dashboards
- [ ] Establish visual hierarchy
- [ ] Choose color scheme (red for bad, green for good)
- [ ] Add drill-down capabilities

## Implementation

- [ ] Build executive dashboard first
- [ ] Configure metric calculations
- [ ] Set up data connections
- [ ] Create filters (by time, segment, etc.)
- [ ] Build drill-down reports
- [ ] Configure alerts and notifications

## Testing & Validation

- [ ] Validate metric calculations
- [ ] Verify data accuracy
- [ ] Test performance
- [ ] Check mobile responsiveness
- [ ] Validate user access controls
- [ ] Get stakeholder sign-off

## Rollout & Training

- [ ] Schedule dashboard training
- [ ] Document dashboard usage
- [ ] Train exec team
- [ ] Train department heads
- [ ] Create FAQ documentation
- [ ] Set up support process

## Maintenance & Optimization

- [ ] Monitor dashboard performance
- [ ] Track user adoption
- [ ] Gather user feedback
- [ ] Optimize slow queries
- [ ] Update metrics as business changes
- [ ] Plan quarterly review cycles
