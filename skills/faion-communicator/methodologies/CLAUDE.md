# Methodologies Folder

## Overview

Contains 10 communication methodologies covering dialogue, validation, conflict, feedback, persuasion, storytelling, negotiation, and brainstorming.

## File Summary

| File | ID | Description |
|------|-----|-------------|
| active-listening.md | active-listening | RASA framework, empathic listening, reflective techniques, paraphrasing, mirroring, open questions |
| mom-test.md | mom-test | Rob Fitzpatrick's validation: 3 rules, good vs bad questions, commitment over compliments |
| stakeholder-communication.md | stakeholder-communication | 5 dialogue modes (Interview, Brainstorm, Clarify, Validate, Socratic), stakeholder mapping |
| conflict-resolution.md | conflict-resolution | Thomas-Kilmann 5 modes, Nonviolent Communication (NVC) 4-step formula |
| feedback.md | feedback | SBI/SBII model, Radical Candor, EEC for positive feedback, LEARN for receiving |
| selling-ideas.md | selling-ideas | SPIN Selling, Challenger Sale, elevator pitch formula, objection handling |
| business-storytelling.md | business-storytelling | Pyramid Principle, SCQA framework, Pixar formula, Hero's Journey |
| negotiation.md | negotiation | Principled negotiation, BATNA/ZOPA, Cialdini's 6 persuasion principles |
| difficult-conversations.md | difficult-conversations | Crucial Conversations 7 skills, DESC script, STATE path |
| brainstorming-ideation.md | brainstorming-ideation | SCAMPER, Mind Mapping, 6-3-5 Brainwriting, Reverse Brainstorm, facilitation |

## Methodology Structure

Each file follows a consistent format:

```
Metadata (ID, Category, Difficulty, Tags, Skill)
Problem (what issue it solves)
Framework (step-by-step process)
Examples (practical applications)
Templates (ready-to-use formats)
Common Mistakes (what to avoid)
Related Methodologies (connections)
References (sources)
```

## Usage by Situation

| Situation | Methodology |
|-----------|-------------|
| Gathering requirements | active-listening + stakeholder-communication |
| Validating business ideas | mom-test |
| Team disagreements | conflict-resolution |
| Performance reviews | feedback |
| Pitching to stakeholders | selling-ideas + business-storytelling |
| Contract discussions | negotiation |
| Sensitive topics | difficult-conversations |
| Generating ideas | brainstorming-ideation |
