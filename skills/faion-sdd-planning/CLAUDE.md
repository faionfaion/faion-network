# faion-sdd-planning Sub-Skill

**SDD Planning Phase: Specifications, Design Documents, Implementation Plans, Task Creation.**

---

## Entry Point

Invoked via [faion-sdd](../faion-sdd/SKILL.md) orchestrator or [faion-net](../faion-net/CLAUDE.md).

---

## Overview

Planning phase of Specification-Driven Development. Creates all documentation artifacts:
- Specifications (WHAT + WHY)
- Design documents (HOW)
- Implementation plans (TASKS)
- Task files (TASK_*.md)
