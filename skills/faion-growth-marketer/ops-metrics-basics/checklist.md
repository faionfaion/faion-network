# Checklist - Operational Metrics Basics

## Define Business Metrics

- [ ] Identify primary success metric (Revenue, Users, Engagement)
- [ ] Choose North Star Metric
- [ ] Define key result areas (KRAs)
- [ ] Set quarterly business objectives
- [ ] Decompose to team-level metrics
- [ ] Document metric ownership

## Revenue Metrics

- [ ] Track Monthly Recurring Revenue (MRR)
- [ ] Measure Average Revenue Per User (ARPU)
- [ ] Calculate Customer Lifetime Value (LTV)
- [ ] Track Customer Acquisition Cost (CAC)
- [ ] Measure LTV:CAC ratio
- [ ] Monitor bookings/pipeline

## User Metrics

- [ ] Track total users
- [ ] Calculate monthly active users (MAU)
- [ ] Measure daily active users (DAU)
- [ ] Track DAU/MAU ratio
- [ ] Measure new user addition rate
- [ ] Monitor user churn

## Engagement Metrics

- [ ] Define core engagement action
- [ ] Track usage frequency (daily, weekly, monthly)
- [ ] Measure session length
- [ ] Track feature adoption
- [ ] Calculate engagement score
- [ ] Monitor engagement trends

## Financial Metrics

- [ ] Calculate Unit Economics
- [ ] Track burn rate (for startups)
- [ ] Measure runway
- [ ] Calculate payback period
- [ ] Track margin by segment
- [ ] Monitor cash position

## Process Metrics

- [ ] Track sales cycle length
- [ ] Measure deal close rate
- [ ] Monitor support response time
- [ ] Measure customer satisfaction (NPS/CSAT)
- [ ] Track onboarding time
- [ ] Monitor product update cadence

## Dashboard Creation

- [ ] Create executive dashboard (3-5 metrics)
- [ ] Build department dashboards (10-15 metrics)
- [ ] Set up real-time metric tracking
- [ ] Configure alerts for anomalies
- [ ] Create weekly reporting views
- [ ] Build historical trend analysis

## Review Process

- [ ] Establish weekly metric review cadence
- [ ] Assign metric ownership
- [ ] Create action items from variances
- [ ] Document insights and learnings
- [ ] Share results across organization
- [ ] Celebrate metric improvements
