# Checklist - Activation Tactics

## Quick Wins (Week 1)

- [ ] Reduce onboarding flow steps (target: 3-5 steps max)
- [ ] Add welcome email with first action link
- [ ] Create in-app tooltip at key moment
- [ ] Simplify sign-up form (name + email only)
- [ ] Remove optional fields
- [ ] Create "Get started" button above the fold

## Personalization (Week 2)

- [ ] Segment users by use case at sign-up
- [ ] Show personalized onboarding flow
- [ ] Customize initial content by user type
- [ ] Personalize email sequence
- [ ] A/B test message variations
- [ ] Test different activation paths

## Engagement Hooks (Week 3)

- [ ] Create interactive tutorial (product walkthrough)
- [ ] Build in-app progress indicator
- [ ] Add streak/streak counter gamification
- [ ] Create milestone celebrations
- [ ] Use contextual help/tooltips
- [ ] Implement smart notifications

## Activation Email Sequence

- [ ] Send welcome email immediately
- [ ] Day 1: Quick start guide
- [ ] Day 2: Feature highlight
- [ ] Day 3: Success story/example
- [ ] Day 5: Offer help/support
- [ ] Day 7: Review checklist

## Remove Friction

- [ ] Eliminate account confirmation step
- [ ] Pre-fill available data (social login)
- [ ] Reduce required integrations at onboarding
- [ ] Simplify initial settings
- [ ] Defer advanced config to later
- [ ] Create skip option for optional steps

## Testing & Iteration

- [ ] A/B test onboarding copy variations
- [ ] Test different flow orders
- [ ] Experiment with activation incentives
- [ ] Test email send times
- [ ] Vary tutorial presentation (video vs text)
- [ ] Test with different user segments

## Measurement & Optimization

- [ ] Track activation rate for each change
- [ ] Measure time to activation
- [ ] Monitor funnel drop-off points
- [ ] Calculate impact on retention
- [ ] Identify remaining friction points
- [ ] Plan next round of improvements
