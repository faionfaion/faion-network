# LLM Prompts

## Prompt 1: Analysis

```
You are a growth marketing expert. Analyze the following [context] and provide:
1. Key insights
2. Recommended actions
3. Potential risks
4. Success metrics

[Context to analyze]
```

## Prompt 2: Implementation Plan

```
Create a detailed implementation plan for [methodology] with:
- Step-by-step tasks
- Resource requirements
- Timeline considerations
- Success criteria
- Risk mitigation

Project context: [details]
```

## Prompt 3: Optimization

```
Review this [metric/campaign/experiment] and suggest optimizations:

Current state:
- [Metrics]
- [Performance data]

Provide:
1. Root cause analysis
2. 3-5 specific optimizations
3. Expected impact
4. Implementation priority
```

## Prompt 4: Troubleshooting

```
Help troubleshoot this issue:

Problem: [description]
Current metrics: [data]
What we've tried: [list]

Provide:
1. Possible causes
2. Diagnostic steps
3. Solution recommendations
4. Prevention strategies
```
