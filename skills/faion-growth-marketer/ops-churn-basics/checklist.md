# Checklist - Churn Analysis Basics

## Define Churn

- [ ] Define customer churn (no purchase in X days)
- [ ] Define user churn (no login in X days)
- [ ] Choose churn period (monthly is standard)
- [ ] Determine observation window
- [ ] Set up churn calculation
- [ ] Document definitions

## Measure Current Churn

- [ ] Calculate monthly churn rate
- [ ] Calculate cohort-specific churn
- [ ] Analyze churn by customer segment
- [ ] Measure churn by acquisition channel
- [ ] Track churn trends over time
- [ ] Benchmark vs industry standards

## Analyze Churn Patterns

- [ ] Identify when users churn (day 1, week 2, month 3)
- [ ] Compare churned vs retained user behavior
- [ ] Analyze feature usage of churners
- [ ] Track customer support interactions
- [ ] Identify product issues in churn users
- [ ] Create churn user personas

## Root Cause Analysis

- [ ] Survey recently churned customers
- [ ] Interview high-value churners
- [ ] Analyze support ticket patterns
- [ ] Review product feedback
- [ ] Identify common pain points
- [ ] Categorize churn reasons

## Churn Segmentation

- [ ] Identify high-risk segments
- [ ] Segment by usage level
- [ ] Segment by company size/revenue
- [ ] Identify geography-specific churn
- [ ] Analyze by product feature adoption
- [ ] Track involuntary vs voluntary churn

## Churn Impact Analysis

- [ ] Calculate revenue impact of churn
- [ ] Determine churn vs customer LTV
- [ ] Analyze cohort lifetime value
- [ ] Calculate payback period by cohort
- [ ] Quantify churn problem
- [ ] Set churn reduction targets

## Prevention Strategy Planning

- [ ] Identify earliest churn signal
- [ ] Design win-back campaigns
- [ ] Plan feature improvements
- [ ] Design support improvements
- [ ] Create onboarding improvements
- [ ] Document improvement roadmap

## Dashboarding & Monitoring

- [ ] Create churn rate dashboard
- [ ] Set up churn alerts
- [ ] Create cohort churn views
- [ ] Build segment-specific churn
- [ ] Track churn reasons
- [ ] Generate weekly churn reports
