# Checklist - Retention Metrics

## Define Active User

- [ ] Determine what "active" means (login, action, purchase)
- [ ] Choose period for activity (daily, weekly, monthly)
- [ ] Segment active status by user type
- [ ] Exclude test/staff accounts
- [ ] Document definition
- [ ] Set up tracking for active status

## Calculate Core Retention

- [ ] Calculate D1 retention (Day 1 / Sign-ups)
- [ ] Calculate D7 retention (Day 7 / Sign-ups)
- [ ] Calculate D30 retention (Day 30 / Sign-ups)
- [ ] Calculate D90 retention (Day 90 / Sign-ups)
- [ ] Measure monthly churn rate
- [ ] Track retention curve

## Create Cohort Analysis

- [ ] Build cohort retention table
- [ ] Organize cohorts by sign-up week/month
- [ ] Track each cohort's retention over time
- [ ] Identify seasonal patterns
- [ ] Compare cohort retention trends
- [ ] Find best/worst performing cohorts

## Segment Retention

- [ ] Analyze retention by acquisition channel
- [ ] Segment by user segment/persona
- [ ] Compare paid vs free users
- [ ] Segment by device/platform
- [ ] Analyze by geographic region
- [ ] Track retention by pricing tier

## Advanced Retention Metrics

- [ ] Calculate Customer Lifetime Value (LTV)
- [ ] Measure engagement frequency (DAU/MAU)
- [ ] Track feature adoption rates
- [ ] Measure session length trends
- [ ] Calculate return user ratio
- [ ] Track reactivation rates

## Predictive Retention

- [ ] Identify early churn signals
- [ ] Create churn prediction model
- [ ] Calculate user health score
- [ ] Flag at-risk users
- [ ] Measure feature engagement by retention group
- [ ] Identify pre-churn behavior patterns

## Dashboarding & Reporting

- [ ] Create retention dashboard
- [ ] Set up daily DAU/MAU tracking
- [ ] Create cohort retention view
- [ ] Configure segment retention views
- [ ] Set up retention alerts
- [ ] Create weekly retention reports

## Benchmarking & Targets

- [ ] Compare vs industry benchmarks
- [ ] Set retention improvement targets
- [ ] Break targets by segment
- [ ] Create quarterly retention goals
- [ ] Establish retention improvement roadmap
- [ ] Track progress vs targets
