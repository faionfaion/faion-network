# Checklist - North Star Metric

## Identification Phase

- [ ] List all possible metrics for your business
- [ ] Interview founders/leadership on business vision
- [ ] Define long-term success for your product
- [ ] Understand business model (freemium, subscription, marketplace)
- [ ] Research what successful competitors measure
- [ ] Brainstorm 5-10 candidate metrics

## Evaluation Phase

- [ ] Test each metric correlates with revenue (for paid products)
- [ ] Verify metric is influenced by product changes
- [ ] Check metric reflects customer value
- [ ] Confirm metric is simple and understandable
- [ ] Ensure metric can be measured accurately
- [ ] Verify metric scales as company grows

## Selection Criteria

- [ ] Does it reflect core value created?
- [ ] Does it correlate with long-term business success?
- [ ] Can it be influenced by product decisions?
- [ ] Is it measurable and trackable?
- [ ] Is it understandable to everyone?
- [ ] Does it align with business model?

## Definition & Measurement

- [ ] Define metric formula precisely
- [ ] Identify numerator and denominator
- [ ] Determine calculation frequency (daily/weekly)
- [ ] Set baseline value
- [ ] Document edge cases and exclusions
- [ ] Create measurement checklist

## Communication & Alignment

- [ ] Present metric to entire company
- [ ] Explain why this metric matters
- [ ] Connect metric to company vision
- [ ] Show how teams can influence the metric
- [ ] Create physical/digital dashboard
- [ ] Update org structure around metric

## Tracking & Dashboarding

- [ ] Set up automated daily tracking
- [ ] Create exec dashboard showing NSM
- [ ] Create detailed drill-down dashboards
- [ ] Set baseline and target values
- [ ] Configure trend analysis
- [ ] Establish weekly review cadence

## Tactical Decomposition

- [ ] Break NSM into key drivers/levers
- [ ] Create AARRR mapping to NSM
- [ ] Identify which team owns each lever
- [ ] Set targets for each driver
- [ ] Create action plans to move NSM
- [ ] Track progress on drivers

## Refinement & Evolution

- [ ] Review metric quarterly for relevance
- [ ] Adjust if business model changes
- [ ] Refine based on learnings
- [ ] Stay consistent (avoid frequent changes)
- [ ] Document metric changes and reasoning
- [ ] Plan evolution as company scales
