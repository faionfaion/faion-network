# Checklist - Activation Metrics

## Define Core Activation Metric

- [ ] Identify key user actions that lead to retention
- [ ] Test correlation between actions and D30 retention
- [ ] Choose metric that differentiates active vs churned users
- [ ] Verify metric is achievable in first session
- [ ] Set activation benchmark (target: 30-50%)
- [ ] Document reasoning for chosen metric

## Set Up Tracking

- [ ] Configure event tracking for activation action
- [ ] Track activation timestamp
- [ ] Segment users by activation status
- [ ] Create funnel from sign-up to activation
- [ ] Set up daily activation tracking
- [ ] Verify tracking accuracy with QA

## Baseline Measurement

- [ ] Calculate current activation rate
- [ ] Measure activation by acquisition channel
- [ ] Analyze activation by user cohort
- [ ] Track time to activation (median/average)
- [ ] Identify drop-off points before activation
- [ ] Create activation cohort table

## Secondary Activation Metrics

- [ ] Track onboarding completion rate (benchmark: 60-80%)
- [ ] Measure time to first value (benchmark: <10 min)
- [ ] Monitor feature adoption rates
- [ ] Track depth of product exploration
- [ ] Analyze activation by user segment
- [ ] Monitor activation over time

## Analysis & Segmentation

- [ ] Compare activation by channel (organic, paid, etc.)
- [ ] Analyze activation by device/platform
- [ ] Segment by user demographic/psychographic
- [ ] Compare activated vs non-activated user behavior
- [ ] Identify pre-activation friction points
- [ ] Create activation cohort reports

## Targets & Goals

- [ ] Set activation rate target (typically +10-20% improvement)
- [ ] Break down targets by channel
- [ ] Set time-to-activation target
- [ ] Define quarterly activation goals
- [ ] Establish activation milestones
- [ ] Align with revenue targets

## Reporting & Dashboards

- [ ] Create daily activation dashboard
- [ ] Set up weekly activation reports
- [ ] Create channel-specific activation views
- [ ] Configure activation alerts/anomalies
- [ ] Track activation trends over time
- [ ] Share metrics with product team
