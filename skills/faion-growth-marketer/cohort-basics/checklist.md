## Understand Cohorts

- [ ] Learn cohort analysis definition

- [ ] Learn retention cohort table format
- [ ] Understand cohort metrics (retention rate, D1/D7/D30)
- [ ] Learn how to read cohort tables
- [ ] Understand cohort trends

## Define Cohorts

- [ ] Choose cohort grouping (weekly or monthly is standard)
- [ ] Define cohort start date
- [ ] Determine analysis window
- [ ] Choose retention metric (active users)
- [ ] Document cohort definition
- [ ] Set up tracking

## Calculate Retention by Cohort

- [ ] Calculate D1 retention for each cohort
- [ ] Calculate D7 retention for each cohort
- [ ] Calculate D30 retention for each cohort
- [ ] Calculate D90 retention (if applicable)
- [ ] Build retention cohort table
- [ ] Analyze cohort table patterns

## Analyze Cohort Performance

- [ ] Compare early vs recent cohorts
- [ ] Identify improving/declining trends
- [ ] Calculate cohort quality metric
- [ ] Analyze cohort curve shape
- [ ] Find cohorts with best retention
- [ ] Identify problematic cohorts

- [ ] Track cohort behavior differences
- [ ] Identify quality issues

- [ ] Identify seasonal patterns
