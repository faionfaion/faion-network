# Checklist - Growth Referral Programs

## Program Design

- [ ] Define referral goal (new users, revenue, engagement)
- [ ] Choose incentive structure (both sides, one side, tiered)
- [ ] Set incentive amounts (test multiple levels)
- [ ] Determine eligible actions (any user, power users only)
- [ ] Design referral flow (sharing moment)
- [ ] Create referral landing page

## Setup & Implementation

- [ ] Choose referral platform (ReferralCandy, Viral Loops, custom)
- [ ] Generate unique referral links
- [ ] Set up tracking for referred users
- [ ] Implement reward dispensing system
- [ ] Configure fraud detection
- [ ] Create referral dashboard

## Promotion & Communication

- [ ] Create referral program landing page
- [ ] Write program email announcement
- [ ] Add referral CTA to product
- [ ] Create in-app referral prompts
- [ ] Design social media campaign
- [ ] Create referral program guides

## Testing & Optimization

- [ ] A/B test incentive amounts
- [ ] Test different reward types
- [ ] Optimize referral landing page
- [ ] Test timing of referral prompts
- [ ] Experiment with messaging
- [ ] Test different sharing channels

## Measurement & Analysis

- [ ] Track referred user count
- [ ] Measure referral conversion rate
- [ ] Calculate cost per referred user
- [ ] Analyze referred user quality/retention
- [ ] Compare referral vs organic users
- [ ] Measure program ROI

## Quality Control

- [ ] Monitor for fraud/gaming
- [ ] Verify referred users are real
- [ ] Check referred user retention
- [ ] Compare retention with organic users
- [ ] Identify quality issues
- [ ] Adjust program parameters if needed

## Scaling & Evolution

- [ ] Increase promotional visibility
- [ ] Add gamification (leaderboards, badges)
- [ ] Create power user ambassador program
- [ ] Implement tiered rewards
- [ ] Expand to new channels
- [ ] Build community around referrals

## Ongoing Management

- [ ] Review program metrics weekly
- [ ] Adjust incentives based on performance
- [ ] Monitor fraud patterns
- [ ] Share success stories
- [ ] Recognize top referrers
- [ ] Plan program improvements
