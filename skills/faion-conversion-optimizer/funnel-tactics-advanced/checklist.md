# Checklist: Funnel Optimization Tactics - Advanced

## Industry Analysis

- [ ] Identify target industry (SaaS, e-commerce, mobile app, other)
- [ ] Research biggest drop-off points for the industry
- [ ] Benchmark conversion rates against industry standards
- [ ] Document current funnel metrics

## Personalization Strategy

- [ ] Segment users by traffic source
- [ ] Segment users by geographic location
- [ ] Segment users by device type
- [ ] Identify returning vs new user segments
- [ ] Define user intent segments
- [ ] Design personalized messaging for each segment
- [ ] Implement personalization rules in analytics
- [ ] Project expected lift (15-40%)

## Exit Intent Implementation

- [ ] Analyze exit intent triggers for your product
- [ ] Design exit intent offer (discount, incentive, etc.)
- [ ] Create compelling exit intent copy
- [ ] Set up mouse leave detection
- [ ] Implement dismissal tracking
- [ ] Test on different browser/device combos
- [ ] Set up conversion tracking for exit intent
- [ ] Measure recovery rate (target: 5-15%)

## Retargeting Campaign Setup

- [ ] Configure email retargeting sequence
- [ ] Set up immediate email (24h post-abandonment)
- [ ] Create follow-up email for 24h+ (offer incentive)
- [ ] Set up ad retargeting (1-7 day window)
- [ ] Design retargeting ad creative with social proof
- [ ] Track conversion lift per channel
- [ ] Measure email recovery (target: 5-10%)
- [ ] Measure ad retargeting recovery (target: 2-5%)

## ICE Prioritization

- [ ] List all optimization hypotheses
- [ ] Score each hypothesis: Impact (1-10)
- [ ] Score each hypothesis: Confidence (1-10)
- [ ] Score each hypothesis: Ease (1-10)
- [ ] Calculate ICE score = Impact × Confidence × Ease
- [ ] Prioritize tests: 20-30 (immediate), 15-20 (this quarter), 10-15 (if resources)
- [ ] Document scoring rationale

## Measurement & Tracking

- [ ] Set up funnel event tracking
- [ ] Track Homepage Viewed event
- [ ] Track CTA Clicked event (with location)
- [ ] Track Signup Form Viewed event
- [ ] Track Signup Completed event (with method)
- [ ] Track Onboarding Started event
- [ ] Track First Action Completed event
- [ ] Calculate overall conversion rate
- [ ] Calculate step-by-step conversion rates
- [ ] Track drop-off rates per step
- [ ] Monitor time-to-convert metric

## Testing & Validation

- [ ] Run A/B test on top priority hypothesis
- [ ] Collect minimum sample size for statistical significance
- [ ] Analyze test results (p-value < 0.05)
- [ ] Document winning variation
- [ ] Implement winning variation
- [ ] Monitor post-implementation metrics
- [ ] Plan next test based on results
