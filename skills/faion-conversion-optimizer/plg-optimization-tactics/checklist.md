# Checklist: PLG [Methodology]

## Phase 1: Activation & Value

- [ ] Define aha moment (what creates value)
- [ ] Reduce time to value (TTV)
- [ ] Streamline onboarding experience
- [ ] Show value before asking for commitment
- [ ] Measure activation rate (target: 20-30% improvement)

## Phase 2: Free Tier Strategy

- [ ] Design free tier (features, limits, duration)
- [ ] Make free tier valuable on its own
- [ ] Create clear upgrade incentives
- [ ] Define paid tiers clearly
- [ ] Test free tier limits and features
- [ ] Track free-to-paid conversion (target: 3-5%)

## Phase 3: Self-Serve & Checkout

- [ ] Optimize checkout flow (1-2 steps max)
- [ ] Minimize form fields and requirements
- [ ] Add multiple payment options
- [ ] Clear pricing visibility
- [ ] Implement abandoned checkout recovery
- [ ] Measure checkout completion (target: 70%+)

## Phase 4: Usage-Based Growth

- [ ] Track key usage metrics
- [ ] Define expansion triggers (usage milestones)
- [ ] Automate upsell at right moment
- [ ] Create upgrade paths (free → pro → enterprise)
- [ ] Test expansion messaging
- [ ] Measure revenue per customer growth

## Phase 5: Metrics & Monitoring

- [ ] Track time-to-value (TTV)
- [ ] Monitor feature adoption rates
- [ ] Measure conversion funnel health
- [ ] Create weekly metrics dashboard
- [ ] Segment by cohort/source
- [ ] Alert on declining metrics

## Phase 6: Test & Iterate

- [ ] A/B test onboarding variations
- [ ] Test pricing and tier structure
- [ ] Optimize checkout experience
- [ ] Implement top 3 improvements
- [ ] Document learnings and wins
- [ ] Plan next optimization cycle
