# Checklist

## Implementation

- [ ] **Fat models, thin views** - Business logic in services, views handle HTTP only
- [ ] **Explicit imports** - Use aliases for cross-app imports to prevent circular dependencies
- [ ] **Separation of concerns** - Services for DB writes, utils for pure functions
- [ ] **Type hints everywhere** - All function parameters and returns are typed
- [ ] **Constants over magic strings** - Use TextChoices and constants.py
