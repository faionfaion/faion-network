Query the Plausible Stats API for [site_id] and produce a weekly summary report.
Metrics: visitors, pageviews, bounce_rate, visit_duration.
Period: last 7 days vs previous 7 days (two separate API calls: 7d and 7d with date offset).
Output: markdown table with columns: Metric | This Week | Last Week | Change %.
Append: top 5 pages by visitors this week.
API key: from environment variable PLAUSIBLE_API_KEY.
