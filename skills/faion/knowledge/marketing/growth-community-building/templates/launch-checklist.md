# Community Launch Checklist

## Pre-Launch
- [ ] Platform selected and configured (channels/spaces set up)
- [ ] Welcome flow designed (what happens when someone joins)
- [ ] Initial content seeded: 5-10 conversations planted before first invitation
- [ ] Community guidelines written and posted
- [ ] Moderation plan documented (who, when, escalation path)
- [ ] First 50 invitation list finalized (existing engaged contacts only)
- [ ] Three core rituals defined with cadence

## Launch Week
- [ ] Personal welcome sent to each new member within 48 hours
- [ ] First daily ritual established and running
- [ ] At least one engagement post per day from you
- [ ] All questions and posts responded to within 4 hours
- [ ] Feedback collected from early members on what to improve

## First Month
- [ ] 3+ rituals running consistently
- [ ] Member introduction thread active (new members introducing themselves)
- [ ] First member-generated content (member posted unprompted)
- [ ] Community guidelines refined based on first moderation decisions
- [ ] 3-5 ambassador candidates identified (most active, most helpful)
- [ ] Health metrics baseline established (DAU/MAU, post rate, retention)
- [ ] First ritual variation applied (same structure, different wording)
