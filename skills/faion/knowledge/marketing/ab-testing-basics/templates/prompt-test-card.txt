Required output fields (refuse if any cannot be determined from context):
- hypothesis: if/then/because format
- primary_metric: name + definition (must tie to a business outcome)
- mde_relative: minimum detectable effect as a decimal (e.g. 0.10 = 10% relative lift)
- sample_per_variant: integer (use statsmodels NormalIndPower, alpha=0.05, power=0.80)
- duration_days: integer (sample_per_variant * 2 / daily_traffic)
- guardrails: list of 3+ metrics that must not regress
- end_date: YYYY-MM-DD based on today + duration_days
- kill_criteria: conditions under which to stop early (guardrail breach only)

Rules:
- Refuse if MDE is missing or unspecified.
- Refuse if primary metric is a vanity metric (pageviews, impressions).
- Do not pick a default MDE — ask for one.
- Convert MDE to absolute change explicitly in output.
