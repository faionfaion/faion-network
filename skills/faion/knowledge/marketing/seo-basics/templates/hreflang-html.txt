# hreflang link tags for HTML head
# Rules: self-referencing required, bidirectional, ISO 639-1 codes, x-default required

<head>
  <!-- Self-referencing hreflang (required on every language version) -->
  <link rel="alternate" hreflang="en" href="https://example.com/en/article/" />
  <link rel="alternate" hreflang="uk" href="https://example.com/uk/article/" />
  <link rel="alternate" hreflang="de" href="https://example.com/de/article/" />
  <!-- x-default: language selector page or most common fallback -->
  <link rel="alternate" hreflang="x-default" href="https://example.com/article/" />

  <!-- Regional variants (optional) -->
  <link rel="alternate" hreflang="en-US" href="https://example.com/us/article/" />
  <link rel="alternate" hreflang="en-GB" href="https://example.com/gb/article/" />
</head>

# Critical rules:
# 1. Every page must include the self-referencing tag (its own hreflang)
# 2. If page A links to page B, page B must link back to page A (bidirectional)
# 3. Use ISO 639-1 codes: en, uk, de, fr (not "english", "ukrainian")
# 4. Never combine hreflang with noindex on the same page
# 5. Metadata (title, description) MUST match the page language
