# Landing Page Conversion Audit Checklist

## Above the Fold (20 points)
- [ ] Headline is clear in 5 seconds — can a stranger state what the product does?
- [ ] One primary CTA — no competing buttons above the fold
- [ ] Visual shows the product (screenshot, mockup, or demo)
- [ ] Social proof visible near or below the CTA button

## Copy (20 points)
- [ ] Benefits over features — every claim passes the "So what?" test
- [ ] Specific numbers used (not "significant improvement" but "40% improvement")
- [ ] Customer language, not jargon — could your ICP read this without a glossary?
- [ ] Scannable — headers, bullets, short paragraphs under 3 sentences

## Trust (20 points)
- [ ] Real testimonials with photos and full names
- [ ] Company logos of known customers (B2B) or user count (B2C)
- [ ] Security or compliance badges if handling sensitive data
- [ ] Money-back guarantee or risk reversal in final CTA

## Technical (not copy — run Lighthouse separately)
- [ ] Loads in under 3 seconds on mobile
- [ ] Mobile responsive — above-fold elements correct on 375px width
- [ ] All forms functional and tracking events fire
- [ ] Analytics tracking confirmed (pageview, CTA click, form submit)
