# SaaS Landing Page Structure

## Navigation
[Logo] — [minimal nav: 1 link max] — [CTA button]

## Hero (above the fold)
Headline: [Outcome] + [Timeframe] + [Without Pain Point] (6-12 words)
Subheadline: Expand on headline, add specificity (1-2 sentences)
CTA button: [Action verb] + [outcome] (e.g., "Start Free Trial", "Get Early Access")
Visual: Product screenshot or dashboard showing the core value
Social proof line: "Trusted by X,000+ [ICP descriptor]" — placed directly below CTA button

## Logos (B2B only)
4-6 recognizable customer logos

## Problem Section
[Pain point headline]
[Consequence 1 — with number if possible]
[Consequence 2 — with number if possible]
[Transition: "There's a better way."]

## Solution
"Introducing [Product]"
[Benefit 1] — [icon]
[Benefit 2] — [icon]
[Benefit 3] — [icon]
(Benefits, not features. Use "So what?" test on every feature.)

## How It Works
Step 1: [User action] ([time])
Step 2: [Product action] (automatic)
Step 3: [User outcome] (instant)

## Features
6 features in 2x3 grid: [icon] + [title] + [1-sentence benefit description]

## Testimonials
3 testimonials with real photos. Format:
"[Specific result] in [timeframe]. Before [Product], I struggled with [problem]. Now [positive outcome]."
— [Full Name], [Title] at [Company]

## Pricing (optional)
3 tiers max. Highlight recommended tier. Show annual savings. Money-back guarantee badge.

## FAQ
5-7 conversion objections (not usage questions):
- Can I cancel anytime?
- Is my data secure?
- Do you offer refunds?
- How is this different from [competitor]?
- Do I need a credit card to start?

## Final CTA
[Benefit reminder headline]
[CTA button — same text as hero]
[Risk reversal: 30-day money-back / no credit card / free trial]

## Footer (minimal)
[Links: Privacy, Terms, Contact]
