# App Store Listing Checklist

## iOS App Store

### Metadata
- [ ] Name (30 chars max): [text — Brand + Primary Keyword]
- [ ] Subtitle (30 chars max): [key benefit or secondary keywords]
- [ ] Keywords field (100 chars, comma-separated, no spaces, no duplicates of name/subtitle): [text]
- [ ] Description (4000 chars): [text]
- [ ] What's New: [text]

### Visuals
- [ ] App Icon: 1024x1024 PNG
- [ ] Screenshots 6.5" (1284x2778): 3–10 screens with text overlay
- [ ] Screenshots 5.5" (1242x2208): same set
- [ ] App Preview Video (optional): 15–30 seconds, no signup required to see value

### Ratings and Support
- [ ] Age rating configured
- [ ] In-app review prompt implemented (SKStoreReviewRequest API)
- [ ] Review response templates ready
