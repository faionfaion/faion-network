# Fine-tuning Evaluation Prompt Templates

## Post-training evaluation (format + quality scoring)
Fine-tuned model: {ft_model}. Base model: {base_model}.

For each test prompt below, run both models and score responses on:
- format_match (0 or 1): does the response follow the required output format?
- content_quality (1-5): is the response accurate and helpful?

Test prompts: {test_prompts_json}

Return JSON:
{
  "ft_avg_format_match": float,
  "base_avg_format_match": float,
  "ft_avg_quality": float,
  "base_avg_quality": float,
  "failing_examples": [{"prompt": str, "ft_response": str, "issue": str}]
}
