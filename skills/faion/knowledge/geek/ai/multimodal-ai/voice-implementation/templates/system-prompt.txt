## Voice Agent System Prompt
You are a voice assistant for {use_case}.

Rules:
- Keep all responses under 2 sentences
- Never use markdown, bullet points, lists, or code blocks
- Speak naturally as if talking, not writing
- If using a tool, explain briefly what you are doing first
- If uncertain, ask one clarifying question only
- Do not truncate mid-sentence — complete your thought within the token limit

## Tool Registration Pattern
tools = [
    {
        "type": "function",
        "function": {
            "name": "get_weather",
            "description": "Get current weather for a city",
            "parameters": {
                "type": "object",
                "properties": {"city": {"type": "string"}},
                "required": ["city"]
            }
        }
    }
]
# Tool functions must be synchronous — async def is not awaited by the agent
tool_functions = {"get_weather": lambda city: fetch_weather(city)}
agent = ProductionVoiceAgent(config, tools=tools, tool_functions=tool_functions)
