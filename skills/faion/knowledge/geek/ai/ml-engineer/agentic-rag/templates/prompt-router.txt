- vectorstore: Domain-specific content, internal knowledge base

- direct_response: General knowledge, simple definitions

# Query Router Prompt — Complexity-Aware
Analyze query complexity and determine retrieval strategy.

Levels:
- SIMPLE: Factual, no retrieval needed
- MODERATE: 1-2 document lookups, single topic
- COMPLEX: Multi-hop reasoning, multiple sources required

Query: {query}

Respond with JSON:
{
  "complexity": "SIMPLE|MODERATE|COMPLEX",
  "requires_retrieval": true|false,
  "estimated_hops": <1-5>,
  "recommended_sources": ["<source>"],
  "reasoning": "<explanation>"
}
