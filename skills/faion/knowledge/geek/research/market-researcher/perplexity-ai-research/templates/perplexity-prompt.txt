# perplexity-prompt.txt — XML prompt for Perplexity query decomposition

<perplexity_research>
  <main_question>{{main_research_question}}</main_question>
  <sub_queries>
    <query>{{sub_query_1}}</query>
    <query>{{sub_query_2}}</query>
    <query>{{sub_query_3}}</query>
  </sub_queries>
  <recency>{{hour|day|week|month|year}}</recency>
  <output>Markdown with inline citations. Flag any claim with a single source.</output>
</perplexity_research>

## Variables
- {{main_research_question}} — the full research question to decompose
- {{sub_query_N}} — narrow sub-queries (3-5 total); each targets one dimension:
  market size, key players, recent funding, product features, pricing, trends
- {{recency}} — use "month" for fast-moving markets; "year" for stable markets

## Notes
- Each sub-query should be answerable independently
- Set search_recency_filter to match the recency field
- Use sonar-large for sub-queries, sonar-huge for final synthesis only
- Record citations from every sub-query response; validate URLs before report delivery
