# tool-selection-prompt.txt — XML prompt for tool selection and comparison tasks

## Tool Selection Template
<tool_selection>
  <task>{{task_description}}</task>
  <phase>{{research_phase}}</phase>
  <constraints>
    <budget>{{free|mid|enterprise}}</budget>
    <api_required>{{true|false}}</api_required>
  </constraints>
  <approved_tools>{{comma_separated_approved_list}}</approved_tools>
  <output>Ranked tool list with: tool name, phase fit, API available (yes/no), cost tier, notes</output>
</tool_selection>

## Tool Comparison Template
<tool_comparison>
  <candidates>{{json_array_of_tool_names}}</candidates>
  <criteria>["real-time updates", "API access", "NLP query support", "pricing"]</criteria>
  <output_format>markdown table</output_format>
</tool_comparison>

## Variables
- {{task_description}} — what research task will be performed
- {{research_phase}} — phase from: exploration, competitor-intel, interview-analysis,
  survey-analysis, sentiment, synthesis, synthetic-research
- {{approved_tools}} — explicit approved tools list; agents must not recommend outside this list
- {{json_array_of_tool_names}} — e.g. ["Crayon", "Contify", "Klue"]
