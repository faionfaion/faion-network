# research-prompt.txt — XML prompt templates for research and synthesis tasks

## Research Task Template
<research_task>
  <question>{{research_question}}</question>
  <sources>{{comma_separated_sources}}</sources>
  <output_format>{{output_format_description}}</output_format>
  <verify>cross-reference at least 2 sources per tool claim</verify>
</research_task>

## Synthesis Task Template
<synthesis_task>
  <raw_findings>{{findings_json}}</raw_findings>
  <instruction>Identify contradictions. Flag any claim supported by fewer than 2 sources.
  Output structured markdown report with inline citations.</instruction>
</synthesis_task>

## Variables
- {{research_question}} — the specific research question to answer
- {{comma_separated_sources}} — tools to query (e.g. perplexity, web-search, serpapi)
- {{output_format_description}} — desired output shape (e.g. JSON, markdown table)
- {{findings_json}} — JSON array of {source, content} objects from dispatch step
