# LoRA Config Selection Prompt
# Use to select rank, target_modules, and learning_rate for a given model and task.
# Requires response_format={"type": "json_object"}.

Given model family "{model_family}" and task type "{task_type}", select the optimal LoRA configuration.

Task type options: classification, extraction, instruction_following, style_transfer, coding, math_reasoning

Consider:
- Rank should match task complexity (8 for simple, 16 for general, 32 for complex)
- target_modules must use the EXACT module names for this model family
- learning_rate should be lower for alignment tasks (DPO/ORPO) vs. SFT

Return JSON:
{
  "rank": int,
  "alpha": int,
  "target_modules": ["module_name", ...],
  "learning_rate": float,
  "include_mlp_layers": bool,
  "rationale": "string"
}
