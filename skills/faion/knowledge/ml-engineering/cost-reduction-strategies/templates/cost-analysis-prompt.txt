# Cost Analysis Prompt
# Use to classify existing LLM call sites for optimization opportunities.
# Requires response_format={"type": "json_object"}.

Given this list of LLM call sites with their usage statistics, classify each for cost optimization.

Call sites:
{call_sites_json}

Each call site has: endpoint name, avg_prompt_tokens, avg_response_tokens, calls_per_day,
current_model, temperature.

For each call site, determine:
- cacheable: temperature=0 AND responses are deterministic (same input → same output)
- model_downgradeable: task is simple (classification, extraction, formatting) → cheaper model viable
- batch_eligible: no real-time requirement → OpenAI Batch API (50% discount, 24h latency)

Return JSON:
{
  "analysis": [
    {
      "endpoint": "string",
      "strategy": "cacheable|model_downgradeable|batch_eligible|none",
      "estimated_monthly_saving_usd": float,
      "rationale": "string",
      "risk": "low|medium|high"
    }
  ],
  "total_monthly_saving_usd": float
}
