# Production Metrics Alert Prompt
# Requires response_format={"type": "json_object"}.

Analyze production evaluation metrics from the past 24 hours and compare to the 7-day baseline.

Current 24h metrics:
- non_empty pass rate: {non_empty_rate}
- reasonable_length pass rate: {reasonable_length_rate}
- no_error_patterns pass rate: {no_error_patterns_rate}
- avg output length (chars): {avg_output_length}
- total evaluated requests: {total_evaluated}

7-day baseline:
- non_empty: {baseline_non_empty}
- reasonable_length: {baseline_reasonable_length}
- no_error_patterns: {baseline_no_error_patterns}
- avg output length: {baseline_avg_length}

Flag any check where the absolute drop exceeds 5 percentage points.
Return JSON:
{
  "alerts": [
    {"check": "check_name", "current": float, "baseline": float, "delta": float, "severity": "warning|critical"}
  ],
  "overall_health": "ok|degraded|critical",
  "recommendation": "string"
}
