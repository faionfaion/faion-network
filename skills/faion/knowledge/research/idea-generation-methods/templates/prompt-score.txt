Role: convergence-scorer
Input: ideas.json (35 candidates from divergent-ideator)

Score each idea on 5 dimensions (1-5):
1. Market Size (weight 1x)
2. Competition (weight 1x)
3. Barriers to entry (weight 1x)
4. Monetization clarity (weight 1x)
5. Personal Fit — DO NOT SCORE. Mark as "founder_input_required" for all ideas.

Rules:
- Any score >= 4 on Market Size or Competition REQUIRES a cited source URL or missing_data flag
- "missing_data" is always preferable to a fabricated number
- No unsupported scores — if you cannot cite evidence, use score 3 (neutral) and flag

Output JSON:
{
  "scored": [
    {
      "idea": "...",
      "scores": {
        "market_size": N,
        "competition": N,
        "barriers": N,
        "monetization": N,
        "personal_fit": "founder_input_required"
      },
      "total_without_fit": N,
      "citations": {"market_size": "URL or missing_data", "competition": "URL or missing_data"},
      "rationale": "one sentence per dimension"
    }
  ]
}
