# Scorecard QA Checklist — Pre-Submit Review

Complete before submitting scorecard to the debrief channel.

## Evidence Completeness

For each competency scored:
- [ ] At least one specific STAR example is documented (Situation + Action + Result minimum)
- [ ] No impression-only language ("seemed confident", "good communicator", "liked them")
- [ ] Every score >= 4 has a verbatim quote or concrete described action as evidence
- [ ] Every score <= 2 has a specific missing component documented (e.g., "No quantified Result")

## Protected-Class Review

- [ ] No mention of age, family status, pregnancy, religion, disability, nationality
- [ ] No note about accent, communication style, or "fit" without behavioral evidence

## Hypothetical Acceptance Check

- [ ] No score based on hypothetical answers ("I would...", "I generally...")
- [ ] All scored examples are confirmed past behavior ("I did...", "Last year...", etc.)

## Pronoun Accuracy

- [ ] For any score on Action: candidate used "I" statements, not only "we"
- [ ] If "we" was dominant: probing question was asked and individual contribution documented

## Completeness

- [ ] All competencies have a score AND an evidence note
- [ ] Hire/No Hire recommendation is present
- [ ] Key strengths section has at least one specific example
- [ ] Areas of concern section is filled if score <= 3 on any competency

## Submission Rule

- [ ] Scorecard submitted BEFORE reading any other interviewer's feedback
- [ ] Score is not changed after seeing peer scores (unless new information arises in debrief)
