## Scoring Guide

- **Impact:** How much will this improve the target metric? 1=minimal, 10=transformational
- **Confidence:** How likely is this to work based on diagnosis? 1=guess, 10=very confident
- **Ease:** How easy to implement? 1=very hard, 10=trivial
- **ICE Score:** Impact x Confidence x Ease

## Priority Tiers

- 20-30: Test immediately
- 15-20: Test this quarter
- 10-15: Test if resources available
- Below 10: Backlog
