# AARRR Pirate Metrics

## Summary

**One-sentence:** A five-stage growth measurement framework — Acquisition, Activation, Retention, Revenue, Referral — that provides a complete picture of the user journey.

**One-paragraph:** A five-stage growth measurement framework — Acquisition, Activation, Retention, Revenue, Referral — that provides a complete picture of the user journey. Each stage has one key metric, a benchmark range, and a diagnostic question. The framework forces prioritization: identify the weakest stage and fix it before scaling the others.

## Applies If (ALL must hold)

- Setting up a growth measurement system for a new product or team.
- Weekly/monthly business reviews where a single dashboard must cover the full funnel.
- Diagnosing why revenue growth has stalled despite healthy acquisition.
- Onboarding a new growth hire or co-founder to shared growth vocabulary.

## Skip If (ANY kills it)

- Pre-launch with zero users — benchmarks and funnels require real cohort data to be meaningful.
- Products with complex multi-sided markets where a single AARRR stack misrepresents both sides.
- When you need deep cohort or segmentation analysis — AARRR is a summary frame, not a substitute for cohort tools.
