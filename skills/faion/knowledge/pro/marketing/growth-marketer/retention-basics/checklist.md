# Checklist - Retention Basics

## Understand Retention

- [ ] Define what "active" means for your product
- [ ] Determine retention periods (D1, D7, D30, D90)
- [ ] Calculate current retention rates
- [ ] Create retention cohort table
- [ ] Segment retention by acquisition source
- [ ] Identify your product's retention curve

## Set Baseline Metrics

- [ ] Calculate D1 retention (benchmark: 40-60%)
- [ ] Calculate D7 retention (benchmark: 20-30%)
- [ ] Calculate D30 retention (benchmark: 10-15%)
- [ ] Measure monthly churn rate (benchmark: <5%)
- [ ] Track cohort retention trends
- [ ] Document baseline in dashboard

## Analyze Retention Patterns

- [ ] Create retention curve visualization
- [ ] Identify drop-off periods (week 1, month 2, etc.)
- [ ] Compare retention by user segment
- [ ] Analyze retention by acquisition channel
- [ ] Segment by user type/persona
- [ ] Find cohorts with best retention

## Root Cause Analysis

- [ ] Interview churned users (why did they leave?)
- [ ] Survey at-risk users
- [ ] Analyze usage patterns of retained vs churned
- [ ] Identify common drop-off events
- [ ] Map journey of power users vs churners
- [ ] Identify key retention inflection points

## Create Retention Strategy

- [ ] Define engagement goals post-activation
- [ ] Identify key aha moments
- [ ] Design habit-forming features
- [ ] Plan communication strategy
- [ ] Create win-back campaigns
- [ ] Set retention improvement targets

## Identify Retention Drivers

- [ ] List features used by retained users
- [ ] Correlate features with retention
- [ ] Identify features avoided by churners
- [ ] Find frequency thresholds (how often to use)
- [ ] Identify power user behaviors
- [ ] Create engagement ladder

## Planning Improvements

- [ ] Design engagement email campaigns
- [ ] Create in-app messaging strategy
- [ ] Plan habit-forming features
- [ ] Design notification strategy
- [ ] Create user segments for targeting
- [ ] Plan community/social features

## Measurement & Monitoring

- [ ] Create retention dashboard
- [ ] Set alerts for retention drops
- [ ] Track retention vs activation correlation
- [ ] Monitor cohort trends
- [ ] Weekly retention review
- [ ] Compare vs industry benchmarks
