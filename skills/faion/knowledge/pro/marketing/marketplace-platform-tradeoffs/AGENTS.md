# Freelance Marketplace Platform Tradeoffs

## Summary

**One-sentence:** Upwork / Toptal / Contra / Arc / Braintrust / Malt comparison — fee structure, lead quality, exclusivity, ranking algorithm; pick the right platform for positioning + rate band.

**One-paragraph:** Freelancers waste months on the wrong platform. No methodology compares them or guides platform selection from positioning + rate band. Output: platform decision matrix + entry plan + churn triggers.

## Applies If (ALL must hold)

- freelancer or solo consultant evaluating platform presence
- current rate band $30-$400/hour OR equivalent project-fee
- willingness to invest 4-8 weeks ramp time

## Skip If (ANY kills it)

- freelancer already has full pipeline from referrals — platforms add overhead
- agency size ≥5 — platforms are individual-keyed
- consultant whose positioning requires direct-only (no broker)

## Prerequisites

- current rate band + target rate band
- positioning niche (matches agency-niche-positioning)
- portfolio with 3-5 recent case studies
