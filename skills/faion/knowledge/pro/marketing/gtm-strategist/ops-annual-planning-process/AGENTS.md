# Annual Planning Process

## Summary

**One-sentence:** A structured five-stage planning framework for solopreneurs: Reflect (year-in-review with cited metrics), Envision (12-month vision statement), Strategize (max 3 priorities — hard cap enforced), Plan (annual → quarterly → monthly task tree), Execute (weekly/monthly/quarterly review rhythm).

**One-paragraph:** A structured five-stage planning framework for solopreneurs: Reflect (year-in-review with cited metrics), Envision (12-month vision statement), Strategize (max 3 priorities — hard cap enforced), Plan (annual → quarterly → monthly task tree), Execute (weekly/monthly/quarterly review rhythm). The non-negotiable rule: cap annual priorities at 3 and enforce a quantified kill-switch and leading indicator for each key result.

## Applies If (ALL must hold)

- December/January: annual review + goal setting
- End of each quarter: quarterly review + next-quarter plan
- First week of each month: monthly check-in against quarterly goals
- When metrics show meaningful drift from targets (>15% off expected)
- Decomposing an annual goal into a monthly/weekly task tree

## Skip If (ANY kills it)

- Setting the founder's vision — agents produce regression-to-mean prose; founders write vision, agents critique and operationalize
- Strategic pivots based on new market signal — qualitative, narrative, requires founder ownership
- Personal goals (energy, satisfaction, burnout) — agents have no telemetry on these
- Salary, equity, or capital-allocation decisions inside the plan — financial-planning domain
- Org-wide planning at more than 20 people — use PM-agile or PM-traditional methodologies instead
