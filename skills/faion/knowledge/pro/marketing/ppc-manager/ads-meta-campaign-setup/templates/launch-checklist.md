# Meta Campaign Launch Checklist

## Campaign: [Name]

### Pre-Launch
- [ ] Meta Pixel installed and verified (green in Pixel Helper)
- [ ] Conversion events firing with correct values in Events Manager
- [ ] Business Manager connected to Instagram Business Account (if using Instagram)

### Campaign Settings
- [ ] Objective selected matches actual business goal
- [ ] Conversion event specified (not link click)
- [ ] Budget type selected: CBO or ad-set
- [ ] Daily budget: $20+ minimum
- [ ] Campaign name follows naming convention

### Ad Set Settings
- [ ] Audience size 500K-5M (check with Audience Estimator)
- [ ] Purchasers excluded from prospecting ad sets
- [ ] Placements: Advantage+ (start) or manual with correct creative sizes
- [ ] UTM parameters added to destination URLs
- [ ] Optimization goal aligned with campaign objective

### Ad Settings
- [ ] 3+ creative variations per ad set
- [ ] Mix of formats: image, video, or carousel
- [ ] Primary text written (125 chars visible)
- [ ] Headline written (40 chars max)
- [ ] CTA button selected
- [ ] Preview checked on Feed, Stories, and Reels

### Launch
- [ ] All items above checked
- [ ] Campaign set to ACTIVE (or scheduled start)
- [ ] Calendar reminder: do not edit for 7 days
- [ ] Calendar reminder: check learning phase status at day 7
