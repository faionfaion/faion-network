## Problem

You want to reach tech-savvy audiences and conversations are happening on Twitter/X. But the platform has changed significantly, and advertising here requires understanding its unique dynamics.

Twitter/X ads work for specific use cases: tech products, news, real-time events, and engaged niche communities.

---

## Framework

Twitter/X advertising leverages real-time and interest-based targeting:

```
TARGET    -> Interests, followers, keywords
CREATIVE  -> Native-looking, valuable content
CONVERT   -> Website clicks or app installs
```

### Step 1: Campaign Objectives

**Available objectives:**

| Objective | Best For | Optimization |
|-----------|----------|--------------|
| Reach | Brand awareness | Impressions |
| Engagements | Content promotion | Likes, retweets |
| Followers | Audience building | New followers |
| Website Traffic | Clicks | Link clicks |
| App Installs | Mobile apps | Installs |
| Video Views | Video content | Views |
| Conversions | Sales, signups | Website conversions |

**For startups:** Website Traffic or Conversions are most common.

### Step 2: Targeting Options

**Targeting methods:**

| Method | Description |
|--------|-------------|
| **Interests** | Predefined interest categories |
| **Follower Lookalikes** | Similar to followers of accounts |
| **Keywords** | People who tweeted or engaged with keywords |
| **Conversation Topics** | Trending and ongoing conversations |
| **Events** | Live events and moments |
| **Tailored Audiences** | Your data (website, email, app) |
| **Demographics** | Age, gender, location |

**Effective combinations:**

| Strategy | Setup |
|----------|-------|
| **Interest-based** | Interests + Demographics |
| **Follower lookalikes** | @competitor followers |
| **Keyword-based** | Keywords people tweet about |
| **Retargeting** | Website visitors |

### Step 3: Ad Formats

**Available formats:**

| Format | Specs | Best For |
|--------|-------|----------|
| **Image** | 1200x675 or 1:1 | Standard promotion |
| **Video** | 0:15-2:20, max 512MB | Engagement, demos |
| **Carousel** | 2-6 images/videos | Multiple products |
| **Text only** | 280 chars | Simple messages |
| **App card** | Image + app install | Mobile apps |

**Creative best practices:**
- Look native (not overly polished)
- Strong first impression (no slow builds)
- Clear value proposition
- Conversational tone

### Step 4: Write Twitter Ads

**Ad copy structure:**

```
[Hook - stop the scroll]
[Value - what they get]
[CTA - what to do]
[Link]
```

**Example formats:**

**Problem/Solution:**
```
Tired of [problem]?

We built [product] to help you [outcome].

Try it free: [link]
```

**Social proof:**
```
10,000+ [audience] are using [product] to [benefit].

Join them: [link]
```

**Direct value:**
```
Free [resource] for [audience]:

- [Benefit 1]
- [Benefit 2]
- [Benefit 3]

Get it here: [link]
```

### Step 5: Budget and Bidding

**Cost expectations:**
- CPM: $5-15
- CPC: $0.50-3.00
- Cost per follower: $2-5

**Budget recommendations:**
- Minimum: $20/day
- Recommended: $50+/day for testing

**Bidding strategies:**
- Automatic bid (recommended start)
- Maximum bid (control max CPC)
- Target bid (optimize for target cost)

### Step 6: Measurement

**Key metrics:**

| Metric | Good | Great |
|--------|------|-------|
| Engagement rate | 1% | 3%+ |
| CTR | 0.5% | 1%+ |
| CPE | <$0.50 | <$0.25 |
| CPC | <$2 | <$1 |

**Tracking setup:**
- Install Twitter pixel
- Set up conversion events
- Use UTM parameters

## Examples

### Example 1: SaaS Product

**Targeting:**
- Follower lookalikes: @stripe, @productboard, @linear
- Interests: Startups, SaaS, Technology
- Location: US

**Ad:**
```
Building a product but drowning in feature requests?

We built a simple way to collect, prioritize, and ship what users actually want.

Used by 500+ product teams.

Try free for 14 days: [link]
```

**Results:**
- CTR: 0.8%
- CPC: $1.50
- Trial signups: 45
- Cost per signup: $35

### Example 2: Newsletter Promotion

**Targeting:**
- Keywords: "startup newsletter," "tech newsletter"
- Interests: Entrepreneurship, Venture Capital
- Follower lookalikes: @TechCrunch, @ProductHunt

**Ad:**
```
Every Friday, 25,000 founders get:

- 1 actionable growth tactic
- 3 tools worth trying
- 5 trending opportunities

Join for free: [link]
```

**Results:**
- CPC: $0.80
- Cost per subscriber: $1.50
- 2,000 new subscribers

## Common Mistakes

| Mistake | Why It Fails | Fix |
|---------|--------------|-----|
| Overly promotional | Ignored by users | Native, valuable content |
| Wrong targeting | Irrelevant audience | Use follower lookalikes |
| Too polished | Doesn't feel authentic | Natural, conversational |
| No testing | Can't optimize | 3+ variations |
| Ignoring replies | Missed engagement | Monitor and respond |
