# LLM Prompts

## Code Generation Prompts

### Generate Implementation
```
Generate [component type] following [methodology name] methodology.

Requirements:
- [Requirement 1]
- [Requirement 2]

Constraints:
- [Constraint 1]
- [Constraint 2]

Output: Complete implementation with tests and documentation.
```

## Review Prompts

### Code Review
```
Review this code against [methodology name] best practices:

[CODE]

Check for:
- Compliance with methodology standards
- Common anti-patterns
- Performance issues
- Security concerns

Output: Detailed review with specific improvement suggestions.
```

## Refactoring Prompts

### Refactor to Methodology
```
Refactor the following code to follow [methodology name] methodology:

[CODE]

Target improvements:
- [Improvement 1]
- [Improvement 2]

Output: Refactored code with explanation of changes.
```

## Documentation Prompts

### Generate Documentation
```
Generate documentation for this [component type] following [methodology name] standards:

[CODE]

Include:
- Overview and purpose
- Usage examples
- Configuration options
- API reference (if applicable)

Output: Complete documentation in Markdown format.
```
