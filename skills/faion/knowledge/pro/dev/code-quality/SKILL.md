# Code Quality Domain Skill

## Purpose

Pro-tier deep-dive into enterprise-grade architecture patterns. Extends free-tier `faion-code-quality` with deeper coverage of DDD, CQRS, event sourcing, and microservices design for teams building complex domain models and distributed systems.

## When to Use

| Scenario | Methodology |
|----------|-------------|
| Layered, dependency-inverted design | clean-architecture |
| Read/write segregation, scalable queries | cqrs-pattern |
| Modeling complex domains with bounded contexts | domain-driven-design |
| Audit-friendly, replayable state | event-sourcing-basics |
| Production event store, projections, snapshots | event-sourcing-implementation |
| Decomposing a system into independent services | microservices-design |
