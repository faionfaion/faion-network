# Team Development

## Summary

**One-sentence:** A structured approach to building high-performing teams using Tuckman's five-stage model (Forming, Storming, Norming, Performing, Adjourning), a team charter with working agreements, a skills matrix for gap planning, and retrospective cycles for continuous improvement.

**One-paragraph:** A structured approach to building high-performing teams using Tuckman's five-stage model (Forming, Storming, Norming, Performing, Adjourning), a team charter with working agreements, a skills matrix for gap planning, and retrospective cycles for continuous improvement. The PM's role shifts by stage: directive at Forming, mediating at Storming, delegating at Performing, celebrating at Adjourning.

## Applies If (ALL must hold)

- Forming a new team: producing charter, working agreements, RACI, and skills matrix in one pass
- Diagnosing a struggling team via Tuckman stage assessment with recommended PM actions
- Building a skills-gap plan (training, pairing, or hire decisions)
- Coaching solo-to-team transitions (first-hire integration)
- Generating and synthesising retrospective patterns across multiple sprints

## Skip If (ANY kills it)

- HR/performance-management actions (compensation, PIPs, hiring decisions) — require a human + HR system
- Crisis interventions (harassment, mental health, layoffs) — agents must escalate to humans immediately
- Cross-cultural mediation where confidentiality and nuance dominate — automation backfires
- Single-person projects — no team to develop
