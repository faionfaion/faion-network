# Quick Project Cost Estimate

Use for early-stage scoping when WBS is not yet available.

## Formula

```
Labor = Team size x Avg daily rate x Duration (days)
```

## Example

- Team: 3 people
- Avg rate: $500/day
- Duration: 60 days
- **Labor: $90,000**

Then apply multipliers:

| Add-on | Rate | Amount |
|--------|------|--------|
| Tools/licenses | +7% | $6,300 |
| Infrastructure | +12% | $10,800 |
| Contingency | +20% | $18,000 |
| **Total** | | **$125,100** |

## Accuracy Warning

ROM (Rough Order of Magnitude) estimates carry ±50% uncertainty.
Do not use this for contract pricing or budget approval.
Upgrade to bottom-up WBS estimate once scope is >50% defined.
