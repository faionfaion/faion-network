# UI Designer Domain Skill

## Purpose

Pro-tier UI design coverage for emerging modalities. Extends the solo-tier `ui-designer` skill with deep design-token strategy (W3C standard, semantic tokens, multi-platform distribution), Voice UI design, and Spatial / XR (AR/VR) interaction patterns.
