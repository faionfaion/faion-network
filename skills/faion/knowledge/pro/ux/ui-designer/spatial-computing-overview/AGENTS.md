# Spatial Computing Overview

## Summary

**One-sentence:** Reference overview of spatial computing platforms, terminology, and market context for designers entering AR/VR/MR work.

**One-paragraph:** Reference overview of spatial computing platforms, terminology, and market context for designers entering AR/VR/MR work. Covers the distinction between AR, VR, and MR; the current platform landscape (Apple Vision Pro, Meta Quest, Android XR, HoloLens, Magic Leap); and the design implications of 3D-space interfaces versus rectangular screens.

## Applies If (ALL must hold)

- Onboarding to a spatial computing project with no prior XR design experience
- Choosing which XR platform to target for a given use case
- Writing briefs or technical specs that need precise AR/VR/MR terminology
