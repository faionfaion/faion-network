# Researcher Domain Skill

## Purpose

Solo-tier general-research toolkit for solopreneurs validating problems, users, and value propositions. Covers feature discovery, Jobs-to-be-Done, pain-point research, problem validation, success metrics, use-case mapping, user interviews, and value-prop design.
