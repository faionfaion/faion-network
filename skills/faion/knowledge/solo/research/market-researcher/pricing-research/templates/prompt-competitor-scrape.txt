Role: competitor-scraper
Input: list of competitor pricing page URLs (randomized order — prevents anchoring)

For each URL, extract the pricing table. Return ONLY what is explicitly stated on the page.

Required fields per competitor:
- name: company name
- url: exact URL scraped
- free_tier_exists: true/false
- tiers: [{tier_name, monthly_price, annual_price, included_features: [], limits: {}, notes: ""}]
- sales_led: true if any tier is "contact us" / "call us" / no price shown
- source_quote: exact text from page that supports the price (required for each price point)

Rules:
- Do NOT estimate or interpolate any price — mark as null if not shown
- Do NOT infer features not explicitly listed — mark as []
- Mark sales-led tiers as price: null, not_shown: true
- If page is paywalled or JS-heavy and not renderable, return: {name, url, error: "page_not_accessible"}

Output JSON:
[{
  "name": "...",
  "url": "...",
  "free_tier_exists": bool,
  "tiers": [{
    "tier_name": "...",
    "monthly_price": N or null,
    "annual_price": N or null,
    "included_features": ["..."],
    "limits": {"users": N, "projects": N, ...},
    "sales_led": bool,
    "source_quote": "exact text from page"
  }]
}]
