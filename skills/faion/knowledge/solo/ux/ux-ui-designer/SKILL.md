# UX / UI Designer Domain Skill

## Purpose

Solo-tier UX/UI designer covering both research and design for solopreneurs running the full design loop alone. Covers Nielsen heuristics, information architecture, journey mapping, usability and A/B testing, prototyping, wireframing, mobile UX, design tokens, and design critique. Larger enterprise UX/UI design lives in `faion-pro-ux-ui-designer`.
