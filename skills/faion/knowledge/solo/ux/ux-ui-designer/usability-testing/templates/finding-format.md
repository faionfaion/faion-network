## Severity Reference

| Severity | Criteria | Priority |
|----------|----------|----------|
| Critical | Task blocked AND 60%+ of participants | Fix before any release |
| High | Task blocked OR 60%+ affected | Fix before launch |
| Medium | 40-60% affected, task completed | Fix within 2 sprints |
| Low | Under 40% affected, minor friction | Add to backlog |
