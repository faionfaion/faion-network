# Accessibility Specialist Domain Skill

## Purpose

Solo-tier accessibility coverage centred on the Nielsen usability heuristics most directly tied to accessible design — flexibility, help/docs, real-world match, recognition over recall, and visibility of system status. Larger WCAG / ARIA / assistive-tech depth lives in `faion-pro-accessibility-specialist`.

## When to Use

| Scenario | Methodology |
|----------|-------------|
| Power-user shortcuts and adaptable UI | flexibility-efficiency |
| In-product help and documentation | help-documentation |
| Matching system to real-world concepts | match-real-world |
| Designing for recognition not recall | recognition-over-recall |
| Surfacing system status to users | visibility-of-system-status |
