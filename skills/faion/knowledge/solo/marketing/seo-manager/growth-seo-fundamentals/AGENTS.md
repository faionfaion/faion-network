# SEO Fundamentals

## Summary

The three-pillar SEO framework — Technical (crawlability), On-Page (content and intent match), Off-Page (backlinks and trust) — covering audit checklists, content optimization by search intent, link building strategy overview, keyword research approach, and key metrics for organic growth.

## Why

53% of all website traffic comes from organic search. Without SEO, content is invisible regardless of quality. Most SEO failures trace to one missing pillar: a technically broken site that can't be indexed, content that mismatches search intent, or zero external authority signals. Understanding all three pillars prevents wasted effort on the wrong layer.

## When To Use

- Starting SEO from scratch on a new project
- Diagnosing why a site gets no organic traffic despite publishing content
- Setting up a monthly SEO reporting framework
- Creating an SEO content brief for a target keyword
- Identifying which pillar (technical/on-page/off-page) is the current bottleneck

## When NOT To Use

- When you need deep technical SEO implementation detail — use seo-basics
- When you need advanced link building tactics — use growth-seo-link-building
- For AI search optimization (AEO) — use technical-seo-for-ai or zero-click-search-adaptation
- When PPC is the channel, not organic — use ppc-manager
