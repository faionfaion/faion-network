# SEO & AEO Mastery

## Summary

**One-sentence:** Comprehensive SEO and AEO (Answer Engine Optimization) methodology covering technical SEO, on-page optimization, multilingual hreflang, schema markup, llms.

**One-paragraph:** Comprehensive SEO and AEO (Answer Engine Optimization) methodology covering technical SEO, on-page optimization, multilingual hreflang, schema markup, llms.txt, Core Web Vitals, accessibility, and social meta tags. Optimizes content for both traditional search engines (Google, Bing) and AI assistants (ChatGPT, Perplexity, Claude).

## Applies If (ALL must hold)

- Running a full SEO audit on a new or existing site
- Implementing technical SEO foundations (robots.txt, sitemap, HTTPS, Core Web Vitals)
- Setting up multilingual sites with hreflang
- Adding schema markup (Article, Organization, FAQ, BreadcrumbList, HowTo)
- Creating llms.txt for AI crawler guidance
- Optimizing for AI Overviews, featured snippets, and zero-click answers
- After a Core Web Vitals regression (LCP/INP/CLS thresholds breached in CrUX)
- After a Google algorithm update or sudden traffic drop (≥20 percent week-over-week)

## Skip If (ANY kills it)

- Content strategy decisions (use content-marketer methodology instead)
- Paid search campaigns (use ppc-manager)
- Link building (use growth-seo-link-building)
- When the site has no indexable pages yet — fix technical blockers first
- When the site is intentionally non-indexable (intranet, staging, private app)
