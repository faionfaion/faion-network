# SEO & AEO Mastery

## Summary

Comprehensive SEO and AEO (Answer Engine Optimization) methodology covering technical SEO, on-page optimization, multilingual hreflang, schema markup, llms.txt, Core Web Vitals, accessibility, and social meta tags. Optimizes content for both traditional search engines (Google, Bing) and AI assistants (ChatGPT, Perplexity, Claude).

## Why

Modern traffic comes from two distinct discovery paths: traditional crawl-index-rank and AI-assisted answer generation. Sites optimized for only one path miss the other entirely. The dual-optimization approach ensures visibility in both — the same structural clarity, authoritative content, and proper markup that satisfies Google's E-E-A-T also enables AI crawlers to cite and reference your content.

## When To Use

- Running a full SEO audit on a new or existing site
- Implementing technical SEO foundations (robots.txt, sitemap, HTTPS, Core Web Vitals)
- Setting up multilingual sites with hreflang
- Adding schema markup (Article, Organization, FAQ, BreadcrumbList)
- Creating llms.txt for AI crawler guidance
- Optimizing for AI Overviews and featured snippets

## When NOT To Use

- Content strategy decisions (use content-marketer methodology instead)
- Paid search campaigns (use ppc-manager)
- Link building (use growth-seo-link-building)
- When the site has no indexable pages yet — fix technical blockers first
