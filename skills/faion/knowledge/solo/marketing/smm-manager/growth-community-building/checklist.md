# Community Building Checklist

## Phase 1: Strategy & Foundation

### Business Case & Goals
- [ ] Define why community (retention, feedback, etc.)
- [ ] Set community objectives
- [ ] Define success metrics
- [ ] Identify target members
- [ ] Estimate growth timeline
- [ ] Allocate budget
- [ ] Assign owner/manager

### Platform Selection
- [ ] Compare platform options (Discord, Slack, Circle)
- [ ] Evaluate pros/cons for your use case
- [ ] Consider integration with existing products
- [ ] Test platforms with small group
- [ ] Make final platform choice
- [ ] Set up account/workspace
- [ ] Customize branding

### Community Vision & Culture
- [ ] Define community purpose (clear statement)
- [ ] Document community values
- [ ] Create community guidelines/rules
- [ ] Define member expectations
- [ ] Plan community rituals
- [ ] Document culture in handbook
- [ ] Create onboarding materials

## Phase 2: Seeding & Early Members

### Member Recruitment
- [ ] Identify first 50 engaged customers
- [ ] Invite personally (not blanket email)
- [ ] Explain community value
- [ ] Start with beta/founding members
- [ ] Offer founding member perks
- [ ] Ask for feedback on setup
- [ ] Create sense of exclusivity

### Launch Planning
- [ ] Create launch announcement
- [ ] Plan launch event/space
- [ ] Prepare welcome materials
- [ ] Set up channels/sections
- [ ] Train first moderators
- [ ] Schedule launch activities
- [ ] Create launch timeline

### Initial Setup
- [ ] Create welcome/rules channel
- [ ] Create introductions channel
- [ ] Create general discussion
- [ ] Create support/help channel
- [ ] Create feedback channel
- [ ] Create announcements channel
- [ ] Create resources/library channel

## Phase 3: Content & Rituals

### Regular Content Calendar
- [ ] Daily: Monitor and respond (1-2 hours)
- [ ] Weekly: Plan community content
- [ ] Weekly: Host community event/discussion
- [ ] Bi-weekly: Share valuable content
- [ ] Monthly: Leadership message/update
- [ ] Monthly: Community spotlight
- [ ] Document content calendar

### Community Rituals
- [ ] Weekly discussions (topic-based)
- [ ] Monthly AMAs (Ask Me Anything)
- [ ] Monthly member spotlights
- [ ] Seasonal events/celebrations
- [ ] Pair programming/mentorship sessions
- [ ] Show & tell sessions
- [ ] Learning sessions/workshops

### Value Creation
- [ ] Share exclusive content
- [ ] Provide early access to features
- [ ] Host educational sessions
- [ ] Enable networking
- [ ] Provide expert access
- [ ] Share company insights
- [ ] Create learning resources

## Phase 4: Engagement & Moderation

### Community Management
- [ ] Respond to all messages within 24 hours
- [ ] Recognize and celebrate contributions
- [ ] Tag valuable contributions
- [ ] Welcome new members personally
- [ ] Foster conversations
- [ ] Prevent spam/off-topic
- [ ] Address issues promptly

### Moderation Setup
- [ ] Create moderation team
- [ ] Train moderators
- [ ] Define escalation process
- [ ] Create rules enforcement plan
- [ ] Document decisions
- [ ] Handle conflicts constructively
- [ ] Regular moderation reviews

### Member Activation
- [ ] Profile early lurkers
- [ ] Send personalized invitations to participate
- [ ] Ask questions to draw them in
- [ ] Tag them in relevant discussions
- [ ] Celebrate their first contribution
- [ ] Create onboarding buddy system
- [ ] Track activation metrics

### Sub-Communities

## Phase 7: Retention & Churn Prevention

### Engagement Monitoring
- [ ] Track monthly active members
- [ ] Monitor participation rates
- [ ] Identify at-risk members (no activity)
- [ ] Track sentiment
- [ ] Monitor satisfaction
- [ ] Track net promoter score
- [ ] Conduct regular surveys

### Re-engagement Campaigns
- [ ] Monitor inactive members
- [ ] Send personal check-ins
- [ ] Offer help/support
- [ ] Share relevant content
- [ ] Invite back to community
- [ ] Gather feedback on why inactive
- [ ] Adjust based on feedback

### Retention Initiatives
- [ ] Create member benefits program
- [ ] Offer exclusive perks

- [ ] Enable skill development
- [ ] Create mentorship programs
- [ ] Offer certification programs
- [ ] Build loyalty through value
