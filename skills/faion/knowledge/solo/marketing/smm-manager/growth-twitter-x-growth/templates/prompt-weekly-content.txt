# purpose: LLM prompt that generates the next week's X post queue (3-5/day) from the unique-angle statement.
# consumes: unique-angle string + last-week's outliers
# produces: a 21-35 post queue consumed by the X scheduler
# depends-on: templates/thread-template.md

Content pillars: <pillar_1> (40%), <pillar_2> (30%), <pillar_3> (20%), personal (10%).

   - Mix: opinions, tips, questions, observations

   - Tweet 1: hook (niche-specific, not "I went from X to Y in Z days")
   - Tweet 2: promise
   - Tweets 3-7: value points
   - Tweet 8: summary + CTA

3. 1 build-in-public update (fill-in fields):
   - MRR: [fill]
   - Users: [fill]
   - Churn: [fill]
   - Win: [fill]
   - Loss: [fill]
   - Next goal: [fill]

Format: tweets on separate lines. Mark threads T1, T2, etc. BIP on its own section.
