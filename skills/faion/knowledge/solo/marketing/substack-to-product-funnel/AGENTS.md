# Substack-to-Product Funnel

## Summary

**One-sentence:** Substack-specific monetization: paid tier as product, recommendations, cross-posts — the distinct discipline newsletter-growth doesn't cover.

**One-paragraph:** growth-newsletter-growth is generic. Substack-specific monetization is a distinct discipline; many indie hackers run a Substack as primary distribution moat. Output: Substack-specific funnel + recommendation strategy + paid-tier design.

## Applies If (ALL must hold)

- indie hacker running OR planning to run a Substack
- topic has ≥1,000-subscriber TAM
- founder commits to ≥6 months of consistent publishing

## Skip If (ANY kills it)

- single-product creator with no audience-build interest
- platform-agnostic newsletter (Beehiiv/Ghost) — Substack-specific tactics don't apply
- research / industry niche where Substack network effects are weak

## Prerequisites

- Substack account + publication setup
- publishing schedule (≥weekly)
- first-30-days editorial calendar
