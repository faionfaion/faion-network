# Difficult Conversations

## Summary

The Crucial Conversations framework structures high-stakes conversations (high stakes + differing opinions + strong emotions) through seven sequential skills: Start with Heart → Learn to Look → Make it Safe → Master My Stories → STATE My Path → Explore Others' Paths → Move to Action. The DESC script (Describe + Express + Specify + Consequences) is a shorter format for written boundary-setting. Both require separation of observable facts from interpretations before any message is drafted.

## Why

Avoiding difficult conversations causes festering resentment, repeated problems, damaged trust, and eventual explosion. The most common failure modes are: waiting too long (address within 3-5 days), using email or text instead of face-to-face, bringing up the past instead of the current issue, and failing to end with a specific WHO/WHAT/WHEN commitment. A well-prepared script reduces anxiety and prevents the conversation from derailing into defensive arguments.

## When To Use

- Preparing a script for a performance issue conversation separating observable facts from interpretations
- Drafting a DESC-structured written message to set a boundary (e.g., after-hours messaging, scope creep)
- Converting a reactive emotional draft into a STATE-structured, fact-based opening
- Running through the preparation checklist before a high-stakes conversation (saying no to a boss, peer conflict, systemic problem)

## When NOT To Use

- Active crisis conversations requiring real-time de-escalation — agents cannot adapt to live emotional cues
- HR formal disciplinary processes where documented language must follow legal requirements
- Conversations involving significant power asymmetry where organizational political context is required
- Anonymous feedback or 360-review contexts where protecting identity is the goal
