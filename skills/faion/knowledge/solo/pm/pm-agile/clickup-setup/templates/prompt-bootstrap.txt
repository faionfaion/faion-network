Config: <config_json>

Steps:
1. Create Space: <space_name> with statuses from config
   POST /team/{team_id}/space
2. Create Folders: <folder_list> inside the space
   POST /space/{space_id}/folder
3. Create Lists: <list_map> inside each folder
   POST /folder/{folder_id}/list
4. Create custom fields at Space level: <fields>
   POST /space/{space_id}/field

Authorization header: "<TOKEN>" (no Bearer prefix).
Rate limit: 100 req/min. Add 700ms delay between requests.
Return created IDs for all objects as JSON.
