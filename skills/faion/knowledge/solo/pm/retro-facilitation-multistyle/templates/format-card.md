# Retro Formats — Selection Card

## mad-sad-glad
Fastest to run. Use when team is fresh + in-person + retro is the first or second of the cycle. Skip when distribution is async (low-signal anonymous responses).

## 4ls (Liked, Learned, Lacked, Longed For)
Use mid-cycle when the team wants more reflective categories than mad-sad-glad. Good after launches; balances positive + critical.

## sailboat (anchors / wind / rocks / island)
Visual metaphor; use when team needs to surface blockers + aspirations. Strong on hybrid; can do anonymous-async on a digital whiteboard.

## lean-coffee
Use when topic-list is unknown — team votes the agenda. Best with stable in-person teams. Skip when shy contributors get bulldozed.

## anonymous-async
MANDATORY for distributed async teams (UA/PT/UK timezones, P4 outsource). Submit cards anonymously via form/board; facilitator clusters; live discussion happens at recap. Skip only if team is fully co-located.

## Selection rule (one-liner)

```
distribution == async              → anonymous-async
fatigue == fatigued                → rotate to a format not used in the last 2 retros
fatigue == fresh + in-person       → mad-sad-glad
mid-cycle + reflective need        → 4ls
visual metaphor + blockers focus   → sailboat
agenda is unknown                  → lean-coffee
```
