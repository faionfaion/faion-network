# Content Audit Spreadsheet — Column Schema

## Identification

| Column | Type | Description |
|--------|------|-------------|
| URL | Text | Full page URL |
| ID | Text | Internal content ID |
| Title | Text | Page or content title |

## Metadata

| Column | Type | Description |
|--------|------|-------------|
| Author | Text | Content owner |
| Date Published | Date | Original publication date |
| Date Updated | Date | Last modification date |
| Content Type | Select | Blog, Product page, Help article, Landing page, Other |

## Taxonomy

| Column | Type | Description |
|--------|------|-------------|
| Category | Text | Primary topic category |
| Tags | Text | Comma-separated tags |

## Status

| Column | Type | Description |
|--------|------|-------------|
| Status | Select | Published, Draft, Archived |

## Performance Metrics

| Column | Type | Description |
|--------|------|-------------|
| Pageviews (90d) | Number | Last 90 days page views |
| Bounce Rate | Percent | Session bounce rate |
| Avg Time on Page | Duration | Average session duration |

## Evaluation

| Column | Type | Description |
|--------|------|-------------|
| Accuracy Score | 1-5 | Is information current and correct? |
| Relevance Score | 1-5 | Does it serve user needs? |
| Quality Score | 1-5 | Is it well-written and complete? |
| Overall Score | Formula | Average of the three scores |
| Action | Select | Keep, Update, Consolidate, Rewrite, Remove, Review |
| Priority | Select | High, Medium, Low |
| Notes | Text | Specific issues or decisions |
