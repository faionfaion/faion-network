# Sunday roadmap & week-shaping ritual

## Intent

Friday-clear backlog: top 3 outcomes for the week chosen, single-tasked file/board updated, no-go items deferred, Monday-morning first task primed with prompt + acceptance.

## Scope

Friday-clear backlog: top 3 outcomes for the week chosen, single-tasked file/board updated, no-go items deferred, Monday-morning first task primed with prompt + acceptance.

## Stages

### 1. Inbox + backlog sweep

Clear the noise so signal is visible.

Tasks:
- Triage inbox and tracker into one place
- Drop everything older than 30 days that nobody has chased
- Tag remaining items with revenue-relevance

Outputs:
- Triaged backlog
- Dropped-item log
- Revenue-relevance tags

Decision gate: Advance only when inbox is empty and backlog has <=20 active items.

### 2. Pick top 3 outcomes

Force ranking. Three or fewer outcomes per week.

Tasks:
- List candidate outcomes from backlog
- Score by RICE for one-person
- Commit to the top 3 with explicit non-goals

Outputs:
- Top 3 outcomes
- Non-goals list
- RICE score sheet

Decision gate: Advance only with exactly 3 named outcomes. No and also.

### 3. Prime Monday

Remove all friction from Monday morning startup.

Tasks:
- Open the first task file with prompt + acceptance criteria
- Cue any external dependencies (data pulls, replies awaited)
- Set the calendar block for first deep work session

Outputs:
- Monday-ready task file
- Calendar block
- Pre-fetched dependencies

Decision gate: Advance when first task can start without any look-up at 9am Monday.

### 4. Deferred & dropped log

Make trade-offs visible so they do not haunt.

Tasks:
- Move non-top-3 candidates to backlog or dropped
- Note the why for each defer
- Set a re-evaluation date if relevant

Outputs:
- Deferred log
- Dropped log
- Re-evaluation reminders

Decision gate: Advance only when every candidate has a written disposition.

### 5. Friday-clear handshake

Promise yourself the week ends on Friday.

Tasks:
- Set a Friday checkpoint slot on the calendar
- List the acceptance criteria for this week is shipped
- Pre-commit to no work on Saturday unless emergency

Outputs:
- Friday checkpoint slot
- Weekly acceptance criteria
- Boundary contract with self

Decision gate: Closes when the ritual file is saved and dated.

## Common pitfalls

- Skipping the decision-gate write-up to keep moving - closes the loop with vibes, not evidence.
- Treating each stages outputs as optional - every output is a gate input for the next stage.
- Letting one bad week stretch a fixed-cadence ritual into a quarterly one.

## Quality checklist

- Every stage has at least one referenced methodology that resolves under `knowledge/`.
- Every output is a real artefact, not a feeling.
- The final decision is a written commitment, not we will see.
