# Free / Business Discovery Playbooks

Find a real problem worth solving before you write code. Audience: first-time founders, marketer-turned-builders, side-hustlers searching for a niche. Citation scope: `knowledge/free/` only.

## Playbooks

| Slug | Goal |
|------|------|
| `mom-test-customer-interview` | Run a Mom Test interview that surfaces real evidence |
| `idea-validation-landing-page` | Validate demand with a landing page + small paid traffic |
| `niche-selection-framework` | Pick a niche on the 4-axis scorecard |
| `is-this-a-real-problem` | Triangulate 3 evidence types before treating a problem as real |
