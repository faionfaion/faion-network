# Free / Dev Fundamentals Playbooks

Code basics: language fundamentals, dependency management, testing, formatting, README. Audience: anyone writing code at the beginner level. Citation scope: `knowledge/free/` only.

## Playbooks

| Slug | Goal |
|------|------|
| `python-first-project` | Write and run your first Python script |
| `javascript-first-project` | Build your first JavaScript project that runs in a browser |
| `dotenv-secrets-management` | Keep secrets out of code with `.env` files |
| `python-package-manager` | Manage Python deps with `uv` (recommended 2026) |
| `write-good-readme` | Write a README that gets your repo noticed |
| `testing-intro-python` | Write your first automated test with pytest |
| `code-style-and-prettier` | Auto-format code so it always looks professional |
