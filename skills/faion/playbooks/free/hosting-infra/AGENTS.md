# Free / Hosting Infra Playbooks

Free deploy + DNS playbooks. Audience: beginners setting up their first domain or static site. Citation scope: `knowledge/free/` only.

## Playbooks

| Slug | Goal |
|------|------|
| `buy-domain-namecheap-cloudflare` | Own a domain and delegate DNS to Cloudflare |
| `deploy-static-site-github-pages` | Host a static site free on GitHub Pages |
| `deploy-to-vercel-free` | Deploy a static or Node app to Vercel free tier |
| `cloudflare-dns-free-ssl` | Add DNS records and enable free SSL on Cloudflare |
