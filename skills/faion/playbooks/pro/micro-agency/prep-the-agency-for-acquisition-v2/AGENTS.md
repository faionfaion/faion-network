# Prep the agency for acquisition (or graceful walk-away)

## Context

Get a 2–3 person agency to a state where it is sellable to a strategic buyer, an acqui-hirer, or transferable to a manager-operator within 6–12 months. Even if not selling, the same prep removes founder from the critical path.

## Outcome

By the end of this playbook, the operator has run the 5 stages below and produced the written decision artefact in the final stage.

Success criteria:

- All 5 stages have written outputs in the project record
- Each stage's decision gate was answered before advancing (yes / no in writing)
- Final stage produced the required written decision artifact
- Every methodology reference loaded cleanly via `faion get-content`

## Steps

### 1. Decide the Path

Acquisition vs graceful walk-away — pick one.

Tasks:
- Score: do I want a buyer or do I want to shut down cleanly
- Set the target outcome (sale price, walk-away timeline)
- Tell the founder advisors and start the clock

Outputs:
- path decision
- target outcome memo
- advisors briefed

Decision gate: Advance when the path is written down with a deadline.

### 2. Tighten the Books

A buyer (or your own clean exit) needs clean financials.

Tasks:
- Reconcile last 24 months of revenue, costs, and ownership
- Separate founder personal expenses from agency books
- Produce a clean P&L + balance sheet review

Outputs:
- reconciled 24-mo books
- clean P&L + BS
- advisor review

Decision gate: Advance only when the books pass an independent advisor review.

### 3. Document the Operating System

Make the business runnable without you.

Tasks:
- Document SOPs, client contracts, vendor list, and IP
- Identify single-points-of-failure (you) and write the bench / replacement plan
- Draft an org-and-systems map

Outputs:
- SOP repository
- SPOF + bench map
- org-and-systems map

Decision gate: Advance when a deputy could run delivery for 30 days using these docs.

### 4. Build the Pitch or Wind-Down Plan

Outcome-specific package.

Tasks:
- If acquisition: build the data room and the seller deck
- If walk-away: write the client handover and contract termination plan
- Get advisor signoff on the pitch or wind-down plan

Outputs:
- data room + seller deck OR handover plan
- advisor signoff

Decision gate: Advance only when advisor signs off in writing.

### 5. Execute

Hit the timeline.

Tasks:
- Talk to 5-10 qualified buyers OR run the wind-down per plan
- Negotiate or finalize handovers; close any open scope cleanly
- Sign the deal or sign the agency closed

Outputs:
- buyer conversations log OR handover log
- signed deal OR closure record

Decision gate: Required output: written closure — sold, signed, or shut down.

## Decision points

- Stage 1 (Decide the Path): Advance when the path is written down with a deadline.
- Stage 2 (Tighten the Books): Advance only when the books pass an independent advisor review.
- Stage 3 (Document the Operating System): Advance when a deputy could run delivery for 30 days using these docs.
- Stage 4 (Build the Pitch or Wind-Down Plan): Advance only when advisor signs off in writing.
- Stage 5 (Execute): Required output: written closure — sold, signed, or shut down.
