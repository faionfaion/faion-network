# Cold lead to signed contract (3-week acquisition flow)

## Intent

From first outbound touch on Upwork/LinkedIn to a counter-signed SOW with deposit landing in escrow; pipeline of 3-5 qualified leads in parallel, one close per cycle.

## Scope

From first outbound touch on Upwork/LinkedIn to a counter-signed SOW with deposit landing in escrow; pipeline of 3-5 qualified leads in parallel, one close per cycle.

## Stages

### 1. Niche + positioning

Position as a specialist, not a generalist.

Tasks:
- Lock the freelance niche statement
- Build a positioning one-liner
- Update LinkedIn + Upwork profile copy

Outputs:
- Niche statement
- Positioning one-liner
- Profile updates

Decision gate: Advance only when profile copy passes 3 outside readers.

### 2. Lead sourcing

Build a fresh pipeline of 20 qualified prospects.

Tasks:
- Search Upwork + LinkedIn + niche communities
- Filter by budget, industry, urgency
- Add to a tracked pipeline

Outputs:
- Pipeline of 20 leads
- Lead-scoring criteria
- Tracker

Decision gate: Advance only when pipeline has >=20 qualified leads.

### 3. Cold outreach

Send 20 personalised messages.

Tasks:
- Write a personalised opener per lead
- Reference a specific pain or signal
- Schedule follow-ups at +3 and +7 days

Outputs:
- 20 personalised messages
- Follow-up schedule
- Open + reply log

Decision gate: Advance only when reply rate >=10%.

### 4. Discovery calls

Active-listening calls, not pitch calls.

Tasks:
- Book 30-minute discovery slots
- Run elicitation interviews to surface scope + budget
- Qualify against ideal-client criteria

Outputs:
- 3-5 discovery calls
- Notes per call
- Qualify/disqualify decisions

Decision gate: Advance only with >=3 qualified leads.

### 5. Proposal + SOW

Send a proposal that closes itself.

Tasks:
- Draft a proposal from the template
- Anchor pricing + scope + deliverables
- Include legal MSA/SOW + payment terms

Outputs:
- Proposal doc
- MSA/SOW pack
- Payment + escrow note

Decision gate: Advance only when proposal is sent within 48h of discovery.

### 6. Negotiate + close

Hold price, push deposit + escrow.

Tasks:
- Run negotiation calls with prepared anchors
- Address common objections (price, timeline, team)
- Lock the deposit terms

Outputs:
- Negotiated terms
- Objection-handling notes
- Deposit terms

Decision gate: Advance only when client agrees to deposit in escrow.

### 7. Sign + escrow

Hand-off to delivery starts only on signed + paid.

Tasks:
- Send counter-signed SOW
- Confirm deposit arrives in escrow
- Kick off the project kickoff playbook

Outputs:
- Counter-signed SOW
- Escrow confirmation
- Kickoff scheduled

Decision gate: Cycle closes when signed + funded.

## Common pitfalls

- Skipping the decision-gate write-up to keep moving - closes the loop with vibes, not evidence.
- Treating each stages outputs as optional - every output is a gate input for the next stage.
- Letting one bad week stretch a fixed-cadence ritual into a quarterly one.

## Quality checklist

- Every stage has at least one referenced methodology that resolves under `knowledge/`.
- Every output is a real artefact, not a feeling.
- The final decision is a written commitment, not we will see.
