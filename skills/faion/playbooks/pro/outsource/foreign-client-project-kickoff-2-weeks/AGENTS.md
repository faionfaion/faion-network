# Foreign-client project kickoff (2 weeks)

## Context

This playbook covers the global flow for the persona `p4-outsource-specialist`. It applies whenever the trigger described in the scope below appears in the operator's workflow. The output is a written, auditable artefact pack, not a verbal hand-wave.

## Outcome

Daria joins a brand-new engagement for a US/EU/UK/AU client. By end of week 2 she has: signed SoW, validated requirements baseline, ratified architecture skeleton fitting client conventions, RACI + comms cadence, dev environment matching client stack, first sprint backlog groomed and an inception report sent. Output is reviewer-ready and survives a senior/lead promotion panel.
