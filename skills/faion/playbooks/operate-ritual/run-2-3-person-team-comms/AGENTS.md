# Run 2–3 person team comms without a full-time PM

## Context

Establish the minimum communication rhythm for a 2–3 person agency where the founder is also the PM. Replaces standups, retros, and 1:1s with one weekly sync + structured async + a Friday close-the-books ritual. Outcome: founder spends <4h/week on internal comms.

## Outcome

By the end of this playbook, the operator has run the 3 stages below and produced the written decision artefact in the final stage.

Success criteria:

- All 3 stages have written outputs in the project record
- Each stage's decision gate was answered before advancing (yes / no in writing)
- Final stage produced the required written decision artifact
- Every methodology reference loaded cleanly via `faion get-content`

## Steps

### 1. Set the Operating Cadence

No full-time PM, so cadence does the PM's job.

Tasks:
- Pick 1 sync per week + daily async standup
- Pick 1 shared decision log + 1 backlog source-of-truth
- Write the cadence one-pager and pin it

Outputs:
- cadence one-pager
- decision log set up
- single backlog

Decision gate: Advance only when every team member has acknowledged the one-pager.

### 2. Run It for 2 Weeks

Make the cadence real before tuning it.

Tasks:
- Hold the sync; respect the time-box (≤45 minutes)
- Use the async standup template daily
- Log every decision and every blocker, no exceptions

Outputs:
- 2-week sync notes
- daily standup logs
- decision + blocker log

Decision gate: Advance when the cadence has held for 2 consecutive weeks.

### 3. Diagnose & Tune

Cut anything that's not pulling weight.

Tasks:
- Ask each team member what they would drop, change, or add
- Cut one meeting or one document if usage is low
- Lock the tuned cadence as the new baseline

Outputs:
- team feedback notes
- cut list
- tuned cadence one-pager

Decision gate: Required output: a written tuned cadence the team has agreed to.

## Decision points

- Stage 1 (Set the Operating Cadence): Advance only when every team member has acknowledged the one-pager.
- Stage 2 (Run It for 2 Weeks): Advance when the cadence has held for 2 consecutive weeks.
- Stage 3 (Diagnose & Tune): Required output: a written tuned cadence the team has agreed to.
