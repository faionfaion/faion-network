## Strategy Mode

### Content Strategy Framework

1. **Audience Analysis**
   - Define buyer personas
   - Map customer journey stages (Awareness → Consideration → Decision)
   - Identify pain points and questions at each stage

2. **Content Pillars**
   - Define 3-5 core topics
   - Create topic clusters (pillar + supporting content)
   - Map keywords to pillars

3. **Content Types**
   - Blog posts (educational, thought leadership)
   - Guides & tutorials (how-to content)
   - Case studies (social proof)
   - Landing pages (conversion)
   - Lead magnets (gated content)

4. **Distribution Channels**
   - Organic search (SEO)
   - Email newsletter
   - Social media
   - Communities (Reddit, Discord, etc.)

5. **Success Metrics**
   - Traffic (organic, direct, referral)
   - Engagement (time on page, scroll depth)
   - Conversions (email signups, purchases)
   - Rankings (keyword positions)

## Audit Mode

### Content Audit Process

1. **Inventory** - List all existing content
2. **Analyze Performance** - Traffic, engagement, conversions
3. **Categorize** - Keep, update, consolidate, delete
4. **Identify Gaps** - Missing topics, outdated content
5. **Create Action Plan** - Prioritized improvements

### Content Audit Scoring

| Score | Performance | Action |
|-------|-------------|--------|
| 5 | High traffic, conversions | Keep, promote |
| 4 | Good traffic, low conversions | Optimize CTA |
| 3 | Moderate performance | Update, improve SEO |
| 2 | Low performance | Consolidate or refresh |
| 1 | No traffic, outdated | Delete or redirect |

## Optimize Mode

### SEO Optimization Checklist

For each piece of content:

1. **Title Tag**
   - Include primary keyword
   - < 60 characters
   - Compelling (emotional, curiosity)

2. **Meta Description**
   - Include keyword naturally
   - 120-160 characters
   - Clear value proposition + CTA

3. **Heading Structure**
   - Single H1 with keyword
   - Logical H2/H3 hierarchy
   - Keywords in subheadings

4. **Content Body**
   - Keyword in first 100 words
   - Natural keyword density (1-2%)
   - Related keywords (LSI)
   - Internal links to related content
   - External links to authoritative sources

5. **Images**
   - Descriptive filenames
   - Alt text with keywords
   - Compressed for speed

6. **Technical**
   - Schema markup (Article, FAQ, HowTo)
   - Canonical URL
   - Open Graph tags

## Writing Guidelines

### Tone & Style

- **Clear** - Simple language, short sentences
- **Actionable** - Practical advice, how-to focus
- **Scannable** - Headers, bullets, bold key points
- **Engaging** - Stories, examples, questions
- **Authoritative** - Data, sources, expertise

### SEO Writing Best Practices

1. **Front-load value** - Best content first
2. **Use transitions** - Guide reader through content
3. **Add variety** - Mix paragraphs, lists, quotes
4. **Include examples** - Concrete, relatable
5. **End with action** - Clear next step
